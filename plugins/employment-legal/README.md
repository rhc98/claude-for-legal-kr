# 근로·고용 실무 플러그인 (employment-legal)

한국 근로·고용·인사 실무를 위한 Claude Code 플러그인. 해고·징계의 정당성(근로기준법 §23) 검토,
근로자성 판단, 임금·근로시간(통상임금·주52시간·최저임금), 직장 내 괴롭힘·성희롱 조사, 취업규칙
작성·변경, 휴가·휴직 법정기한 관리.

원본 [`anthropics/claude-for-legal`](https://github.com/anthropics/claude-for-legal)의
`employment-legal` 플러그인(Apache 2.0)을 한국 노동법체계 — **근로기준법·남녀고용평등법·노동조합 및
노동관계조정법·산업재해보상보험법·산업안전보건법·최저임금법·기간제법·파견법** — 로 포팅한 결과물입니다.

**모든 출력은 변호사·노무사 검토 전제의 초안입니다** — 인용된 조문·판례는 국가법령정보 MCP로 검증해 표시하고,
판단해야 할 지점은 `[검토]` 플래그를 답니다. 결정적 행위(해고 통지 발송, 직원·상대방 송부, 에스컬레이션)는
명시적 확인 게이트를 통과해야 합니다.

> **한국에는 임의해고(at-will employment)가 없습니다.** 미국 원본은 "어느 주냐(state)"에 따른 다주
> 관할 인식이 핵심이지만, 한국 노동법은 전국 단일 관할이며 **모든 해고에 근기법 §23의 "정당한 이유"가
> 필요**합니다. 미국의 "주" 축은 한국에서 **상시 근로자 수(headcount) 기준 적용 차등** 축으로 바뀝니다.

## 누구를 위한 도구인가

| 역할 | 주요 워크플로우 |
|---|---|
| **노동·인사 법무 변호사** | 해고·징계 검토, 근로자성 판단, 취업규칙·정책 작성, 조사 |
| **공인노무사** | 임금·근로시간 분석, 취업규칙 신고, 노동위 사건, 산재 |
| **사내 법무·인사(HR)** | 채용·근로계약 검토, 1차 임금·휴가 Q&A, 휴가·휴직 기한 관리 |
| **법무법인·노무법인** | 다중 의뢰인 매터 워크스페이스, 노동 사건 전반 |

## 핵심 한국법

- **근로기준법** — 근로계약·임금·근로시간·해고·징계·휴가의 기본법. 해고는 §23 정당한 이유 + 절차(§26·§27).
- **상시 근로자 수 기준** — 5인 미만 적용 제외, 10인 이상 취업규칙(§93), 30인 노사협의회 등. 적용 게이트.
- **통상임금** — 2024.12.19. 대법원 전합 판결(고정성 요건 폐기, 2025 현행 기준)이 가산수당 산정의 기초.
- **남녀고용평등법** — 성희롱·차별·육아휴직·출산휴가. **노조법** — 단체교섭·부당노동행위·노사협의.
- **산재보험법·산안법·중대재해처벌법** — 업무상 재해·안전보건·경영책임자 의무.
- **기간제법·파견법** — 비전형 고용. 채용 시 개인정보는 PIPA → [`privacy-legal`](../privacy-legal/) 핸드오프.

## 상시 근로자 수가 핵심 변수

검토 스킬은 **먼저 상시 근로자 수를 확인**합니다 — 해고 제한·가산수당·연차·취업규칙 의무 등 다수
규정이 상시 근로자 수 임계점에 따라 적용·면제됩니다. 산정은 다툼이 잦아 `[검토]`로 플래그하고 국가법령정보
현행 시행령에 대조합니다. 이것이 미국 원본의 "주 supplement"의 한국 대응물입니다.

## 첫 실행: 콜드스타트 인터뷰

플러그인이 인터뷰로 학습합니다: 상시 근로자 수·업종·사업장·노조·취업규칙 여부, 영역별 플레이북 입장
(해고·채용·임금·휴가·괴롭힘·근로자성), 에스컬레이션 임계점, 하우스 스타일. 그 후 시드 문서(취업규칙·
표준 근로계약서·최근 해고/징계 메모)를 읽어 하우스 포지션을 학습합니다.

설정은 `~/.claude/plugins/config/claude-for-legal-kr/employment-legal/CLAUDE.md`에 저장되며 플러그인
업데이트와 무관하게 유지됩니다. 휴가 등록부·조사 로그도 같은 폴더에 저장됩니다 — **민감 인사정보이므로
백업·접근통제 필요.**

```
/employment-legal:cold-start-interview
```

## 비고

- **해고 정당성·근로자성·통상임금·상시 근로자 수 산정은 결론이 아니라 `[검토]` 플래그**입니다. 정당성·
  근로자성은 법원·노동위가 판단하며, 임계점·금액은 시행령·고시(변동)가 결정합니다 — 단정하지 않고
  국가법령정보 현행 대조로 라우팅합니다.
- **노동위원회 판정례·고용노동부 행정해석은 국가법령정보에 포함되어 있지 않을 수 있습니다.** 부당해고·부당노동
  행위는 노동위가 1차 forum이므로 인용 시 중앙노동위 사이트(`https://www.nlrc.go.kr`) 직접 검색 또는
  수동 첨부로 보완. `[노동위 판정 — 직접 검증 필요]`. `docs/DATA_GAPS.md` 참조.
- **해외 사업장·외국 근로자·외국 준거법 근로계약은 외국법 overlay**입니다. 한국이 디폴트 관할이며
  외국법 인용은 `[외국법 — 외국 변호사 검증 필요]`로 플래그하고 현지 노동 전문가 자문을 권고합니다.
- 한국법에는 미국식 work-product 독트린(FRCP 26(b)(3))이 없습니다 — 문서 헤더는 비밀유지 의지의
  표시이지 공개거부의 법적 근거가 아닙니다. **공인노무사 작성물은 변호사·의뢰인 비밀유지보다 보호가 약합니다.**
- 채용·인사에서 개인정보가 흐르면 [`privacy-legal`](../privacy-legal/), AI 채용·평가 도구는
  [`ai-governance-legal`](../ai-governance-legal/) 플러그인과 연계됩니다.
