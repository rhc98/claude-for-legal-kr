# /hiring-review

채용 프로세스 및 근로계약서를 한국 노동법 기준으로 검토한다. 근로기준법 §17 근로조건 명시 의무,
기간제법상 2년 한도 및 예외 사유, 수습·시용 설계, 경업금지·비밀유지 조항 리스크, 채용절차공정화법
준수, 개인정보보호법(PIPA) 핸드오프가 핵심 검토 범위다.

## 실행 순서

1. `~/.claude/plugins/config/claude-for-legal-kr/employment-legal/CLAUDE.md` 읽기 — 실무 프로파일
   확인. populate되지 않았으면: "플러그인이 아직 셋업되지 않았습니다. `/employment-legal:cold-start-interview`를
   먼저 실행하세요. 셋업 없이도 검토를 진행하지만, 상시 근로자 수·플레이북 입장이 디폴트로 동작합니다."
2. 활성 매터가 있으면 `matters/<slug>/matter.md` 읽기 — 매터 특화 override 확인(상시 근로자 수,
   단체협약 가중 조항 등).
3. 아래 **검토 유형 판정** 실행.
4. 플레이북 §채용·근로계약 섹션을 읽고 하우스 입장 적용.
5. 해당 검토 워크플로우 실행.
6. 발견 등록부 + 다음 단계 출력.

---

## 1. 매터 컨텍스트

활성 매터가 있으면 출력 상단에:

```
매터: [slug] | 의뢰인: [의뢰인명] | 상시 근로자 수: [N명 — 검토] | 비밀 등급: [표준/강화]
```

활성 매터 없으면 실무 수준 컨텍스트로 운영 (CLAUDE.md의 상시 근로자 수 밴드 사용).

---

## 2. 목적지 점검

검토 산출물을 출력하기 전, 한 줄 확인:

> **이 검토 결과는 어디로 가나요?** 내부 인사팀 검토용 / 경영진 보고용 / 채용 담당자 전달용 /
> 외부 법률 검토용 / 기타

답에 따라 산출물 헤더의 비밀 등급·배포 범위 주석을 조정한다. 채용 과정 산출물에는 대상 근로자
성명·개인정보가 포함될 수 있으므로, 배포 전 정보 최소화 원칙(PIPA §3) 준수를 권고.

역할이 비전문가(변호사·노무사 자격 없음)이면 산출물 헤더에:

> ※ 이 문서는 리서치·내부 검토용으로 작성되었으며, 법적 자문을 구성하지 않습니다. 채용 결정 실행
> 전 변호사·공인노무사 검토를 받으십시오.

---

## 3. 목적

채용 프로세스 및 근로계약서를 검토하여 다음을 달성한다:

- 근기법 §17 근로조건 명시 의무 충족 여부 확인
- 기간제 계약의 2년 한도·예외 사유 적합성 검토
- 수습·시용 기간 설계의 적법성 평가
- 경업금지·비밀유지 조항의 법적 리스크 파악 (판단은 `[검토]`)
- 채용절차공정화법 위반 여부 점검
- PIPA 개인정보 처리 이슈 식별 및 `/privacy-legal` 핸드오프

## 4. 플레이북 로드

`~/.claude/plugins/config/claude-for-legal-kr/employment-legal/CLAUDE.md`의
`### 채용·근로계약` 섹션 읽기:

- **표준 근로계약 입장:** 기간의 정함, 수습/시용 설계, 경업금지 범위·보상 하우스 입장
- **경업금지·비밀유지 입장:** 퇴직 후 경업금지 기간·지역·대상·보상에 관한 하우스 포지션

입장이 `[PLACEHOLDER]`이면: 검토는 진행하되, 하우스 입장 적용 없이 법령 기준으로만 분석하고
출력에 `[하우스 입장 미설정 — 디폴트 적용]` 플래그.

## 5. 검토 유형 판정

`$ARGUMENTS` 또는 사용자가 제공한 자료로 판정:

| 검토 유형 | 트리거 | 이동할 섹션 |
|---|---|---|
| **신규 근로계약서 초안 검토** | `--contract` / 계약서 첨부 | §7 근로조건 명시 체크리스트 → §8 수습·시용 → §9 경업금지 |
| **기존 계약서 리스크 점검** | "기존 계약", "현행 계약" | §6 기간제 확인 GATE → §7 → §8 → §9 |
| **채용 프로세스·채용공고 검토** | `--process` / 공고 첨부 | §10 채용절차공정화법 → §11 개인정보 |
| **경업금지 조항 단독 분석** | `--nca` / "경업금지", "NCA" | §9 경업금지 (상세) |
| **수습·시용 설계 자문** | "수습", "시용", "수습기간" | §8 수습·시용 (상세) |

유형 불명확하면 한 줄 확인:

> "검토 대상이 어떤 것인지 확인해주세요: ① 근로계약서 초안, ② 기존 계약서 점검,
> ③ 채용 프로세스·공고, ④ 경업금지 조항만, ⑤ 수습·시용 설계?"

## 6. 기간제 여부 확인 (GATE)

근로계약서가 검토 대상이면, 가장 먼저 기간 조항을 확인한다.

### 기간의 정함 없음 (무기계약)

→ §7 근로조건 명시 체크리스트로 바로 이동. 기간제 관련 섹션 skip.

### 기간의 정함 있음 (기간제 계약)

**2년 한도 분석:**

- 계약 기간이 총 2년 초과인가? (기간제법 §4 — 반복 갱신 포함 누적 2년)
  - 2년 이하: ✓ 한도 내 → §7로 이동
  - 2년 초과 또는 반복 갱신으로 2년 접근: **무기계약 전환 간주 위험** `[검토]` 🔴

**예외 사유 확인 (기간제법 §4② 각호):**

2년 초과 또는 근접 시, 아래 예외 해당 여부를 하나씩 점검:

| 예외 사유 | 해당 여부 | 비고 |
|---|---|---|
| 사업의 완료 또는 특정 업무 완성에 필요한 기간 | `[검토]` | 업무 성격·완료 시점 특정 필요 |
| 휴직·파견 등으로 결원된 근로자 대체 | `[검토]` | 원래 근로자 복귀 시점까지로 한정 |
| 고령자(55세 이상, 고용상 연령차별금지법) | `[검토]` | 연령 확인 |
| 전문적 지식·기술이 필요하고 박사학위 소지자 | `[검토]` | 박사 학위 + 전문성 요건 |
| 정부의 복지정책·실업대책 사업 종사 | `[검토]` | 사업 근거 확인 |
| 그 밖에 기간제법 시행령 §3 해당 | `[검토]` | 현행 시행령 법망 조회 권고 |

> **법망 조회 권고:** `mcp__beopmang__law get "기간제 및 단시간근로자 보호 등에 관한 법률 제4조"`
> 현행 시행령 예외 목록 확인. 법망 미연결이면 `[모델 지식 — 검증 필요]`.

**반복 갱신 실태:**

기존 계약서 검토 시, 이전 계약 체결 이력을 확인한다:

- 동일 근로자와 반복 갱신 횟수·기간 → 누적 2년 초과 여부 `[검토]`
- 갱신 기대권 형성 여부 (갱신 거절 시 부당해고 유사 법리 적용 가능성) `[검토]`
- 계약 종료 후 단기 공백·재채용 패턴 → 탈법 여부 `[검토]`

> `[모델 지식 — 검증 필요]`: 반복 갱신 탈법 여부는 노동위 판정례 축적 영역 — 법망 미커버 가능.
> 인용 시 판정서 URL·PDF 요청.

## 8. 수습·시용 조항 검토

수습 또는 시용 조항이 있으면 아래를 순서대로 분석한다.

### 8-1. 수습 (試習, probationary skill-building period)

**목적 구분:** 수습은 기술·업무 습득 기간. 시용과 다르므로 계약서 문언 확인.

| 점검 항목 | 내용 | 판정 |
|---|---|---|
| 기간 | 몇 개월? | 3개월 이하 권고. 초과 시 `[검토]` 🟡 |
| 임금 감액 | 최저임금 90% 적용 여부 | 아래 예외 확인 필수 |
| 감액 불가 케이스 | 1년 미만 계약 또는 단순노무직(최저임금법 시행규칙 별표) | `[검토]` — 해당 시 감액 불가 🔴 |
| 감액 근거 명시 | 계약서에 감액 비율·기간 명시 여부 | 미명시 시 최저임금 위반 위험 |

> **법망 조회:** `mcp__beopmang__law get "최저임금법 제5조"` — 수습 감액 요건·단순노무직 제외 현행 조문.

**단순노무직 판단:** 한국표준직업분류상 단순노무직 해당 여부 `[검토]` — 법망 또는 고용부 고시 확인.
`[고용부 행정해석 — 직접 검증 필요]` 태그 권고.

### 8-2. 시용 (試用, probationary fitness evaluation period)

**본채용 거부의 법적 성격:** 시용 기간 본채용 거부는 **해고에 준하는 처분**으로 정당한 이유 필요
(다만 근기법 §23 정규해고보다 완화된 기준 적용 — 대법원 판례) `[검토]`.

| 점검 항목 | 내용 | 위험 |
|---|---|---|
| 기간 | 몇 개월? | 3개월 이하 권고. 과도한 기간은 시용 남용 위험 `[검토]` |
| 평가 기준 명시 | 본채용 거부 사유·평가 기준이 계약서·취업규칙에 명시되어 있는가 | 미명시 → 자의적 거부 시 부당해고 유사 법리 🔴 |
| 평가 절차 | 평가 실시·통보 절차가 있는가 | 절차 불비 → 본채용 거부 정당성 약화 `[검토]` |
| 서면 통지 | 본채용 거부 결정을 서면으로 통지하는가 | 미통지 → §27 위반 위험 🔴 |
| 소명 기회 | 대상 근로자에게 소명 기회 부여 | 절차적 정당성 요소 — 권고 |

> **주의:** 시용 기간 종료 후 추가 연장은 원칙적으로 제한 — 연장이 본채용 거부 회피 수단이 되면
> 부당해고 유사 법리 적용 가능 `[검토]`.

## 9. 경업금지 조항 검토

경업금지 또는 비밀유지 조항이 있으면 아래를 분석한다. **유효/무효를 단정하지 않는다** —
모든 판단은 `[검토]`.

> **기본 원칙:** 퇴직 후 경업금지는 **헌법 §15 직업선택의 자유**를 제한한다. 법원은
> 보호이익·기간·지역·대상·대가의 종합 형량으로 유효성을 판단 — 어느 한 요소만으로 유효/무효를
> 단정하지 않는다.

> **법망 조회:** `mcp__beopmang__case search "경업금지 직업선택의 자유"` — 최근 판례 동향 확인.
> 법망 미연결이면 `[모델 지식 — 검증 필요]`.

### 9-1. 보호이익 특정성

| 점검 항목 | 내용 |
|---|---|

### 9-2. 기간

| 점검 | 내용 |
|---|---|
| 기간 | ___ 개월 / 년 |

### 9-4. 대상 업무

| 점검 | 내용 |
|---|---|

### 9-5. 대가 (보상)

| 점검 | 내용 |
|---|---|

| 평가 | 보상 없음 → 무효 위험 높음 `[검토]` 🔴. 보상 있음: 유효성↑ (금액 상당성도 고려) |

### 9-6. 비밀유지 조항 구분

경업금지와 별도로 비밀유지(NDA) 조항을 확인:

- 비밀 대상 정보의 범위·관리 수준 명시 여부
- 영업비밀보호법 요건 충족 여부 (비밀 관리성·경제적 유용성) `[검토]`
- 퇴직 후 비밀유지는 경업금지와 달리 보상 없이도 유효한 경우 多 — 구분 필요

### 9-7. 종합 유효성 평가


## 11. 개인정보 처리 체크 (PIPA)

채용 과정의 개인정보 처리를 간략히 점검하고, 상세 분석은 `/privacy-legal`로 핸드오프한다.

### 11-2. 민감정보 수집 여부
