# /employment-legal:policy-drafting

## 매터 컨텍스트

실무 수준 CLAUDE.md `## 매터 워크스페이스` 확인. `활성화`가 `✗`(사내 사용자 디폴트)이면
이 단락 skip — 스킬은 실무 수준 컨텍스트 사용, 매터 머시너리 비활성. 활성이고 활성 매터
없으면: "어느 매터입니까? `/employment-legal:matter-workspace switch <slug>` 실행하거나
`practice-level` 말씀." 활성 매터의 `matter.md` 로드. 산출물은
`~/.claude/plugins/config/claude-for-legal-kr/employment-legal/matters/<매터-slug>/outputs/`.
`매터 간 컨텍스트`가 `on`이 아니면 다른 매터 파일 절대 안 읽음.

## 목적지 점검

취업규칙 초안·검토 산출물은 내부 배포 전 목적지 점검 필수. 사용자가 목적지(직원 전체 공지,
고용노동부 제출, 노동조합 송부, 외부)를 명시했으면 비밀유지 서클 내부인지 묻는다. 서클 외부로
보이면 플래그하고 (a) 법무용 비밀유지 버전, (b) sanitize 버전, (c) 둘 다 중 무엇을 줄지 묻는다.
실무 수준 CLAUDE.md `## 공유 가드레일 → 목적지 점검`이 정본.

---

## 목적

취업규칙의 **신규 작성** 또는 **기재사항 점검**을 지원한다.

- **신규 작성** (`--new`) — §93 법정 기재사항을 갖춘 취업규칙 골격 초안. 섹션별로 작업.
- **특정 섹션 초안** (`--section [섹션명]`) — 징계규정·임금규정·휴가규정 등 지정 섹션만.
- **기재사항 점검** (`--check`) — 기존 취업규칙을 업로드하면 §93 법정 기재사항 누락·불충분 여부를 체크리스트로 출력.

산출물은 법무·HR·공인노무사의 검토 기초로 쓰인다 — 결론이 아니라 초안·플래그의 집합이다.

## 플레이북 먼저 로드

`~/.claude/plugins/config/claude-for-legal-kr/employment-legal/CLAUDE.md`를 읽는다.
`## 플레이북` 전체와 `## 상시 근로자 수 적용범위 overlay`를 적용한다. 취업규칙 관련
하우스 입장(징계 기준·임금 구성·근로시간 원칙)이 구성되어 있으면 섹션 초안에 반영한다.

> **침묵의 보완 금지.** 이 스킬에 필요한 조문(근기법 §93·§94·§97, 근로계약법리)을
> 국가법령정보에서 확인하지 못하면 멈추지 않고 `[모델 지식 — 검증 필요]`로 태그하고 진행한다.
>
> **국가법령정보 조회 예시:**
> ```
> mcp__korean-law__get_law_text("근로기준법 제93조")
> mcp__korean-law__get_law_text("근로기준법 제94조")
> mcp__korean-law__get_law_text("근로기준법 제97조")
> ```
>
> **출처 태그(정본은 CLAUDE.md `## 공유 가드레일`):**
> `[국가법령정보]` / `[모델 지식 — 검증 필요]`(디폴트) / `[고용부 행정해석 — 직접 검증 필요]` /
> `[사용자 제공]`. 변호사·노무사 판단 지점은 `[검토]`.

플레이북 해당 섹션이 `[미구성]`이거나 `[PLACEHOLDER]`이면: 일반 한국법 기준으로 진행하되
검토자 메모에 "플레이북 미구성" 기록. 취업규칙 초안은 하우스 입장이 반영될 때 실무에 맞게
작동한다 — `/employment-legal:cold-start-interview`로 구성을 권고한다.

## 상시 근로자 수 게이트

**이 게이트를 통과하기 전까지 §93 체크리스트·초안 작업을 시작하지 않는다.**

실무 프로파일 `## 우리는 누구인가 → 상시 근로자 수`에서 읽는다. 제공되지 않았으면 먼저 묻는다:

> **상시 근로자 수**가 몇 명입니까? — 취업규칙 작성·신고 의무가 이 기준으로 달라집니다.
> `[상시 근로자 수 산정 — 검증 필요]`

### 10인 미만 사업장인 경우

**근기법 §93 작성·신고 의무가 없다.** `[모델 지식 — 검증 필요]`

단, 취업규칙을 **자발적으로 작성·운영할 수 있다**. 이 경우 중요한 주의:

> ⚠️ **10인 미만 자발적 작성 — §94 적용 주의**
> 취업규칙을 작성해 운영하면 §93 신고 의무는 없으나, §94(불이익변경 동의 요건)는
> 사업장 규모와 무관하게 **작성된 취업규칙에 적용**될 수 있습니다. `[모델 지식 — 검증 필요]`
> 이후 내용을 근로자에게 불이익하게 변경하려면 동의 절차가 필요할 수 있습니다. `[검토]`

10인 미만인 경우 체크리스트를 참고로 제공하되, 신고 의무 없음을 명시한다.

### 10인 이상 사업장인 경우

**§93 작성·신고 의무 발생.** `[모델 지식 — 검증 필요]`

> ⚠️ **10인 이상 — §93 신고 의무**
> 취업규칙을 작성해 고용노동부 지방관서에 신고해야 합니다. 신고 없이 운영하면
> 과태료 제재 대상입니다. `[모델 지식 — 검증 필요]` 아래 §93 법정 기재사항 체크리스트를
> 전부 점검합니다.

## §93 법정 기재사항 체크리스트

`[모델 지식 — 검증 필요]` — 국가법령정보에서 근기법 §93 현행 조문 대조 필수.

기존 취업규칙이 제공된 경우 각 항목을 대조한다. 신규 작성인 경우 빠진 항목을 식별해 초안에 포함할 항목 목록으로 사용한다.

| # | 법정 기재사항 | 상태 | 비고 |
|---|---|---|---|
| M-1 | **시업·종업 시각** — 근로 시작·종료 시각 명시 | ✓ / ⚠️ / ❌ | |
| M-2 | **휴게시간** — 법정 휴게 이상 보장·시간 명시 (§54) | ✓ / ⚠️ / ❌ | |
| M-3 | **휴일** — 주휴일·법정 공휴일 등 명시 (§55) | ✓ / ⚠️ / ❌ | |
| M-4 | **휴가** — 연차유급휴가 (§60) 기타 약정휴가 | ✓ / ⚠️ / ❌ | |
| M-5 | **교대제** — 교대근무 해당 시 교대 방법·교대 주기 | ✓ / ⚠️ / N/A | 교대제 운영 시만 |
| M-6 | **임금 결정·계산·지급방법** — 기본급·수당 구성, 산정 방식 | ✓ / ⚠️ / ❌ | |
| M-7 | **임금 지급시기** — 지급일(근기법 §43 매월 1회 이상) | ✓ / ⚠️ / ❌ | |
| M-8 | **승급(승급·승호) 사항** — 적용 시 | ✓ / ⚠️ / N/A | |
| M-9 | **가족수당** — 지급 시 지급 기준·금액 | ✓ / ⚠️ / N/A | 지급 시만 |
| M-10 | **식비·주거·교통비** — 지급 시 지급 기준 | ✓ / ⚠️ / N/A | 지급 시만 |
| M-11 | **퇴직에 관한 사항** — 정년·자진퇴직 절차·퇴직 효력 발생일 | ✓ / ⚠️ / ❌ | |
| M-12 | **퇴직급여제도** — 퇴직금 또는 퇴직연금(근퇴법) 명시 | ✓ / ⚠️ / ❌ | |
| M-13 | **성과급·상여금** — 해당 시 지급 기준·시기 | ✓ / ⚠️ / N/A | |
| M-14 | **안전·보건에 관한 사항** | ✓ / ⚠️ / ❌ | |
| M-15 | **재해부조에 관한 사항** | ✓ / ⚠️ / ❌ | |
| M-16 | **표창·제재(징계) 사항** — 징계 사유·종류·절차 | ✓ / ⚠️ / ❌ | **없으면 징계해고 절차적 정당성 결여 🔴** |
| M-17 | **기타 전 근로자에게 적용될 사항** | ✓ / ⚠️ / N/A | |

**판정 기준:**
- ✓ **있음** — 해당 내용이 명확하게 규정되어 있음
- ⚠️ **불충분** — 규정은 있으나 내용이 포괄적이거나 불명확해 분쟁 가능
- ❌ **누락** — 법정 필수 항목인데 없음 → 🔴 위반 위험

**M-16 특별 주의:** 징계 규정(사유·종류·절차)이 없으면 징계해고 시 절차적 정당성 결여로
부당해고 위험이 커진다. 취업규칙에 징계 근거 없이 해고하면 취업규칙 위반 + 절차 하자
이중 위험. `[모델 지식 — 검증 필요]` → `/employment-legal:termination-review` 연계 권고.

## 섹션별 초안 지원

사용자가 `--section [섹션명]`을 지정하거나, 신규 작성 시 어느 섹션을 먼저 작업할지 묻는다.
각 섹션 초안은 플레이북 하우스 입장 + §93 법정 기재사항 + 실무 표준 구조를 결합해 작성한다.
초안은 `[검토]` 플래그를 포함한 1차 작업 문서 — 변호사·노무사 검토 전 확정하지 않는다.

### A. 징계규정

징계규정이 없는 취업규칙은 징계해고 시 절차적 정당성 결여 위험. `[모델 지식 — 검증 필요]`

**초안 구조:**

```
제 X 장 징계

제 X 조 (징계의 종류)
  징계는 다음 각 호와 같이 구분한다.
  1. 경고·견책
  2. 감봉 (1회 감봉액은 평균임금 1일분의 1/2 초과 금지, 총액은 1임금지급기 임금 총액의 1/10 초과 금지)
  3. 정직 (1개월 이내)
  4. 강등
  5. 해고 (징계해고)

제 X 조 (징계 사유)
  다음 각 호에 해당하는 행위를 한 경우 징계한다.
  1. 근태 관련: 무단결근 [N]일 이상 / 지각·조퇴 상습 / 업무 이탈
  2. 품위·복무 관련: 직장 내 괴롭힘 / 성희롱 / 폭언·폭행
  3. 비위·윤리 관련: 횡령·배임·절도 / 회사 기밀 유출 / 이해충돌
  4. 업무 관련: 고의 또는 중과실로 회사에 손해 / 지시·명령 위반
  5. 기타: 이 규칙·사규에서 정한 금지사항 위반

제 X 조 (징계위원회)
  징계는 징계위원회의 의결로 한다. [구성·정족수는 플레이북에 따름] `[검토]`

제 X 조 (소명기회 부여)
  징계 대상자에게 사전에 징계 사유를 통보하고 소명 기회를 부여한다.
  소명을 거부하거나 기한 내 소명하지 않으면 소명 포기로 처리한다.

제 X 조 (징계의 통보)
  징계 의결 후 [N]일 이내 서면으로 통보한다. 해고의 경우 근기법 §27에 따라
  해고 사유와 해고일자를 서면으로 명시한다.
```

`[검토]` 포인트:
- 징계 사유 목록이 사업장 업종·하우스 입장에 맞는지
- 감봉 한도(근기법 §95) `[모델 지식 — 검증 필요]` — 국가법령정보 §95 조문 대조 필수
- 징계위원회 구성·의결방식 (단체협약·취업규칙 선례 반영 필요)

### B. 임금규정

```
제 X 장 임금

제 X 조 (임금의 구성)
  임금은 다음으로 구성한다.
  1. 기본급: [결정 방식 — 직급별 호봉표 또는 직무급]
  2. 제 수당: 직책수당, 자격수당, [기타 — 플레이북 참조]
  3. 가족수당: [지급 기준 — 해당 시]
  4. 식대·교통비: [지급 기준 — 해당 시]
  5. 상여금·성과급: [지급 시기·기준 — 해당 시]

제 X 조 (임금의 계산·지급)
  임금은 매월 [N]일에 근로자 본인 명의 계좌에 지급한다.
  지급일이 휴일인 경우 전일에 지급한다.

제 X 조 (임금 공제)
  다음을 임금에서 공제한다.
  1. 법정 공제: 소득세, 4대보험 본인 부담분
  2. 서면 동의를 받은 경우: [기타 공제 — §43②]

제 X 조 (연장·야간·휴일 근로 수당)
  연장·야간·휴일 근로에 대해 근기법 §56에 따른 가산수당을 지급한다. `[검토]`
  (5인 이상 사업장에 한함 — 상시 근로자 수 게이트 결과 반영)
```

`[검토]` 포인트:
- 포괄임금제 채택 여부 — 유효성 요건 충족 여부 (`/employment-legal:wage-hour-qa §E` 연계)
- 통상임금 포함 항목 목록 — 2024.12.19. 전합 이후 기준 `[모델 지식 — 검증 필요 / 전합 이후 하급심·행정해석 변화 진행 중]`

### C. 휴가·휴직규정

```
제 X 장 휴가·휴직

제 X 조 (연차유급휴가)
  근기법 §60에 따라 연차유급휴가를 부여한다. `[모델 지식 — 검증 필요]`
  1. 1년 미만 근로자: 개근한 달마다 1일 발생 (최대 11일)
  2. 1년 이상 근로자: 출근율 80% 이상 시 15일, 3년 이상부터 2년마다 1일 가산 (최대 25일)

제 X 조 (연차사용촉진)
  회사는 근기법 §61에 따른 연차사용촉진 절차를 이행할 수 있다. `[모델 지식 — 검증 필요]`

제 X 조 (출산전후휴가)
  출산전후휴가는 근기법 §74에 따라 90일(다태아 120일) 부여한다. `[모델 지식 — 검증 필요]`

제 X 조 (육아휴직)
  육아휴직은 남녀고용평등법 제19조에 따라 허용한다. `[모델 지식 — 검증 필요]`

제 X 조 (경조사 휴가) [약정 휴가 — 법정 아님]
  경조사별 부여 일수는 별표 X와 같다.

제 X 조 (휴직)
  다음 사유로 휴직을 신청하면 [N]개월 이내 허용한다. `[검토]` [사유 목록]
```

`[검토]` 포인트:
- 경조사 휴가·기타 약정휴가는 취업규칙에 포함되면 근로조건이 되어 §94 적용 대상
- 육아기 근로시간단축·배우자 출산휴가 등 최근 개정 반영 여부 `[모델 지식 — 검증 필요]`

### D. 취업규칙 변경 절차 규정

```
제 X 조 (취업규칙의 변경)
  ① 이 규칙을 변경하려면 근기법 §94에 따라 다음 절차를 거친다. `[모델 지식 — 검증 필요]`
  ② 근로자에게 유리한 변경: 근로자 과반수(또는 노동조합)의 의견을 들은 후 변경한다.
  ③ 근로자에게 불이익한 변경: 근로자 과반수(또는 노동조합)의 서면 동의를 받아 변경한다.
  ④ 변경된 취업규칙은 고용노동부에 신고하고 근로자에게 공지한다 (§14).
```

## §94 불이익변경 판단

사용자가 **기존 취업규칙을 변경**하는 경우 이 섹션을 적용한다. 신규 작성이면 skip.

**판단 전제:** 변경 내용이 이전 규정과 비교해 근로자에게 불이익한가?
이 판단은 사실관계 전체를 종합해야 하므로 항상 `[검토]`.

| 변경 유형 | 불이익 여부 | 필요 절차 |
|---|---|---|
| 임금 삭감·수당 폐지 | 불이익 🔴 | 과반수 노동조합 또는 근로자 과반수 **서면 동의** |
| 근로시간 연장 | 불이익 🔴 | 동의 필요 |
| 징계 사유 추가·양형 강화 | 불이익 가능 🟠 `[검토]` | 동의 필요 가능성 高 |
| 복리후생 축소 | 불이익 가능 🟠 `[검토]` | 동의 필요 가능성 高 |
| 일부 개선·일부 불이익 혼재 | 종합 판단 `[검토]` | 종합 비교 후 결정 `[검토]` |
| 근로자에게 유리한 변경 | 유리 🟢 | 의견청취로 충분 (동의 불필요) |

**동의 없이 불이익변경 시:** 해당 변경 부분은 **무효** — 구 취업규칙이 계속 적용된다.
`[모델 지식 — 검증 필요]`

**개별 근로계약 vs 취업규칙 (§97):** 개별 근로계약이 취업규칙보다 유리한 조건을 정한 경우,
개별 계약이 우선 적용된다. `[모델 지식 — 검증 필요]` 변경이 기존 개별 계약보다 불리해지는지
함께 검토. `[검토]`

> 불이익변경 여부 판단과 동의 절차 안내가 주목적이면 `/employment-legal:handbook-updates`
> 스킬이 더 상세하게 지원합니다.

## §93 신고 절차 (10인 이상)

`[모델 지식 — 검증 필요]` — 현행 절차·서류는 고용노동부 민원마당 또는 지방관서에서 확인.

**신고 방법:**
1. **온라인** — 고용노동부 민원마당(minwon.moel.go.kr)에서 전자 신고
2. **방문·우편** — 사업장 소재지 관할 지방고용노동관서 접수

**제출 서류 (확인 필요):**
- [ ] 취업규칙(신규 작성본 또는 변경본) `[검토 — 현행 서식·요건 지방관서 확인]`
- [ ] 근로자 과반수의 의견청취(또는 동의) 증빙 서류
- [ ] 기타 관할 관서 요구 서류

**신고 없이 운영 시:** 과태료 제재 위험. `[모델 지식 — 검증 필요]`
