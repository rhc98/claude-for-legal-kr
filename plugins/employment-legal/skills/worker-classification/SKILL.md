# /employment-legal:worker-classification

## 1. 매터 컨텍스트

실행 전, CLAUDE.md(`${CLAUDE_PLUGIN_ROOT}/CLAUDE.md`) 및 실무 프로파일
(`~/.claude/plugins/config/claude-for-legal-kr/employment-legal/CLAUDE.md`)이 있으면 읽는다.

- **상시 근로자 수 적용범위 overlay** — 5인 미만·5인 이상 여부에 따라 소급 노출 범위가 달라짐
- **플레이북 §도급·근로자성** — 하우스 입장이 있으면 우선 적용
- **에스컬레이션 매트릭스** — 법적 노출 규모가 임계점을 넘으면 즉시 에스컬레이션

프로파일이 없으면 계속 진행하되, 발견 결과에 `[실무 프로파일 미구성 — 일반 디폴트 사용]` 태그를 붙인다.

---

## 2. 목적지 점검

이 스킬이 적합한 사용 사례:
- **근로자성 판단** — 현재 또는 과거 계약 관계가 근로자인지 개인사업자인지 분류
- **오분류 위험 스크리닝** — 채용 전 계약 구조 검토, 고용노동부 조사 대비, 세무조사 대비
- **소급 노출 추정** — 오분류가 확인되거나 다툼이 생길 경우 재정 노출 규모 파악
- **계약 재설계 검토** — 도급→파견→직접고용 전환 경로 탐색

이 스킬이 *적합하지 않은* 사용 사례 (다른 스킬 안내):
- 특정 해고 건 검토 → `/employment-legal:termination-review`
- 임금·통상임금 계산 → `/employment-legal:wage-hour-qa`
- 취업규칙 불이익변경 → `/employment-legal:policy-drafting`

---

## 3. 목적

1. **분류** — 해당 관계가 근로자(근기법 §2①1호)·특수형태근로종사자·독립사업자 중 어디에 해당하는지 체계적으로 평가
2. **오분류 위험 평가** — 고용노동부 조사·노동위원회 구제신청·소송에서 다툼 가능성 및 방어 가능성 판단
3. **소급 노출 추정** — 근로자로 판단될 경우 4대보험·퇴직금·연차·가산수당 소급 범위를 보수적으로 계산

> ⚠️ 이 스킬은 `[검토]` 태그 없이 "이 사람은 근로자다" 또는 "근로자가 아니다"라고 단정하지 않는다.
> 대법원도 종합적 판단을 요구하며 단일 요소로 결론을 내리지 않는다.

---

## 4. 플레이북 먼저 로드

실무 프로파일의 `## 플레이북 § 근로자성·도급` 섹션을 읽는다.

- 하우스 입장(예: "도급 계약은 반드시 법무 검토 후 체결", "IT 프리랜서 계약은 6개월 이상 시 위험 플래그")이 있으면 이 분석 전체에 적용
- 입장이 없으면 한국 노동법 일반 원칙으로 진행

법망 MCP가 연결되어 있으면:

```
mcp__beopmang__law get "근로기준법 제2조"          # 근로자·사용자 정의
mcp__beopmang__law get "파견근로자보호법 제2조"     # 근로자파견 정의
mcp__beopmang__case search "근로자성 종속노동성 2006다49865"
```

미연결이면: 이하 모든 법령 인용과 판례 참조에 `[모델 지식 — 검증 필요]` 태그.

## 5. 분류 대상 특정

분석을 시작하기 전, 대상 관계의 유형을 특정한다. 사용자에게 묻거나 제공된 계약서·사실관계에서 추출:

| 계약 유형 | 전형적 위험 지점 |
|---|---|
| **프리랜서 계약** | 업무지시·근무시간 구속이 있으면 근로자 방향 |
| **도급(업무위탁)** | 사용사업주가 직접 지휘·감독하면 불법파견 가능 |
| **위임** | 전문직(의사·변호사 등)에 많으나 실질 종속성 체크 |
| **파견** | 파견법 적용 여부, 허용 업종·기간 초과 여부 |
| **사내하청** | 도급 실질이 파견인지 — 불법파견 판단 체크리스트로 |
| **플랫폼 종사자** | 플랫폼 종사자 보호법 적용 여부 `[모델 지식 — 검증 필요]` |

> 사용자가 계약서·사실관계를 제공하면 먼저 읽는다. 제공이 없으면 핵심 사실 질문(아래 6)으로 시작.

## 6. 종속노동성 체크리스트

대법원이 종합적으로 고려하는 10개 요소(대법원 2006다49865 등 `[모델 지식 — 검증 필요]`).
각 요소에 대해: **(a) 사실 확인 질문 → (b) 근로자성 지시 방향 → (c) 이 사건 평가 `[검토]`**

> ⚠️ 아래 모든 평가에 `[검토]` 태그가 붙는다. 단일 요소로 결론을 내리지 않는다.

---

### 6.1 업무 내용을 사용자가 정하는지

**사실 확인 질문:**
- 업무 내용·범위·방식을 누가 결정하는가? 계약서에 명시된 업무 범위가 구체적·세부적인가?
- 업무 수행 중 지시가 내려오는가? 형식(이메일·메신저·구두)은 무엇인가?

**근로자성 지시 방향:** 사용자가 업무 내용을 구체적으로 지정·변경 → 근로자 방향

**이 사건 평가:** `[검토]` — 제공된 사실관계 기반으로 기재

---

### 6.2 취업규칙·복무규정 등 적용 여부

**사실 확인 질문:**
- 회사의 취업규칙·사규·행동강령이 계약 당사자에게 적용되는가?
- 회사 내부 교육·복무 규정 준수 의무가 있는가?
- 회사 이메일 계정·사내 시스템 접근 권한이 부여되어 있는가?

**근로자성 지시 방향:** 사내 복무규정 적용·사내 시스템 전면 사용 → 근로자 방향

**이 사건 평가:** `[검토]`

---

### 6.3 업무 수행 과정에서 구체적·개별적 지휘·감독 여부

**사실 확인 질문:**
- 일별·주별 보고 의무가 있는가? 누구에게 보고하는가?
- 특정 관리자의 승인 없이 업무 방식·절차를 변경할 수 있는가?
- 작업 결과물을 제출 전 검토·수정하도록 요청받는가?

**근로자성 지시 방향:** 구체적·개별적 지휘감독(결재라인·일일보고) → 근로자 방향

**이 사건 평가:** `[검토]`

---

### 6.4 근무시간·장소 구속 여부

**사실 확인 질문:**
- 출퇴근 시간이 정해져 있는가? 지각·조퇴 시 보고 의무가 있는가?
- 근무 장소가 지정되어 있는가(사무실·현장)? 재택·외부 근무 자유도는 어느 정도인가?
- 초과근무·야간근무 요구에 응해야 하는가?

**근로자성 지시 방향:** 출퇴근 시간 고정·근무 장소 지정 → 근로자 방향

**이 사건 평가:** `[검토]`

---

### 6.5 업무의 대체성 (본인이 직접 수행해야 하는지)

**사실 확인 질문:**
- 다른 사람을 고용·위임해 업무를 대신 수행할 수 있는가?
- 계약서에 "본인 직접 수행" 조항이 있는가?
- 실제로 대체 인력을 사용한 적이 있는가? 사용사업주의 동의가 필요한가?

**근로자성 지시 방향:** 본인 직접 수행 의무·대체 불가 → 근로자 방향

**이 사건 평가:** `[검토]`

---

### 6.6 비품·원자재·작업도구 소유·제공 여부

**사실 확인 질문:**
- 업무에 필요한 장비(컴퓨터·차량·도구)를 누가 제공하는가?
- 원자재·소모품 비용을 누가 부담하는가?
- 별도 비용 정산(재료비·장비 임차료)이 이루어지는가?

**근로자성 지시 방향:** 사용자가 주요 도구·시설 전부 제공 → 근로자 방향

**이 사건 평가:** `[검토]`

---

### 6.7 보수가 근로 자체의 대가인지 (vs 사업 성과 대가)

**사실 확인 질문:**
- 보수 산정 기준은 무엇인가? 시급·일급·월급인가, 성과물(건당·프로젝트)인가?
- 결과물 미완성 시 보수 전액이 지급되는가?
- 손실·손해에 대해 개인이 부담하는가?

**근로자성 지시 방향:** 시간·노무 제공 자체에 대한 대가(고정급) → 근로자 방향

**이 사건 평가:** `[검토]`

---

### 6.8 기본급·고정급 여부, 근로소득세 원천징수 여부

**사실 확인 질문:**
- 매월 고정 금액을 수령하는가?
- 소득세 원천징수(갑종 근로소득세)가 이루어지는가?
- 4대보험(건강·국민연금·고용·산재)에 가입되어 있는가?

**근로자성 지시 방향:** 고정급 지급·근로소득세 원천징수·4대보험 가입 → 근로자 방향

> 참고: 원천징수를 했다고 해서 근로자성이 자동 인정되거나 부정되지는 않는다. 반대로 사업소득세로 처리했다고 해서 근로자성이 배제되지 않는다 `[검토]`.

**이 사건 평가:** `[검토]`

---

### 6.9 경제적 종속성 (사용자에게만 노무 제공하는지)

**사실 확인 질문:**
- 동시에 다른 사업주에게 노무를 제공하는가? 실제로 그렇게 하는가?
- 계약에 전속 의무 조항이 있는가?
- 수입의 몇 %가 이 사용자에게서 나오는가?

**근로자성 지시 방향:** 전속적·배타적 노무 제공·수입 의존도 높음 → 근로자 방향

**이 사건 평가:** `[검토]`

---

### 6.10 사회보장제도에서 근로자 인정 여부

**사실 확인 질문:**
- 고용보험·산재보험 피보험자로 등록되어 있는가?
- 건강보험·국민연금을 직장 가입자로 취득하였는가?
- 회사가 사용자 분담금을 납부하고 있는가?

**근로자성 지시 방향:** 4대보험 직장 가입·사용자 분담 납부 → 근로자 방향 (역 방향도 경우에 따라 검토)

**이 사건 평가:** `[검토]`

## 8. 특수 유형 체크

### 8.1 특수형태근로종사자 (특고)

다음 직종에 해당하는지 확인(산재보험법·고용보험법 일부 적용 가능) `[모델 지식 — 검증 필요]`:
- 보험설계사, 학습지 교사, 골프장 캐디, 택배기사, 퀵서비스 기사
- 대출모집인, 신용카드 모집인, 대리운전기사
- 방문판매원, 방문점검원, 가전제품 설치·수리원
- 화물차주, 건설기계 운전원, 소프트웨어 프리랜서(법령상 열거 직종 확인 필요)

특고에 해당하면:
- **산재보험법:** 전속성 요건 폐지 여부 및 적용 범위 확인 `[모델 지식 — 검증 필요]`
- **고용보험법:** 고용보험 적용 직종인지 확인
- 특고 보험 적용 = 근로자성 인정이 아님 — 별도 트랙

> 특고 관련 법령 개정 현황은 자주 변경된다 `[모델 지식 — 검증 필요]`. 법망 MCP로 현행 산재보험법
> 시행령 별표를 확인하거나 고용노동부 현행 고시를 직접 조회할 것.

### 8.2 불법파견 판단 (도급 실질이 파견인지)

도급·사내하청 관계가 있을 경우, 아래 체크리스트로 불법파견 여부를 판단 `[검토]`:

**파견 방향 지시 요소:**
- [ ] 사용사업주의 근로자가 도급 노동자에게 직접 업무지시·감독을 함
- [ ] 도급업체에 독자적인 사업조직·기술·전문성이 없음 (인력 공급에 불과)
- [ ] 사용사업주가 채용·해고에 실질적으로 관여함
- [ ] 도급업체 소속 노동자가 사용사업주 근로자와 혼재하여 동일 업무 수행
- [ ] 사용사업주가 근태·휴가·복무를 직접 관리함
- [ ] 도급 계약 목적물이 "업무 수행" 자체이지, 특정 결과물이 아님

불법파견으로 판단되면:
- 파견근로자보호법 §6의2: **사용사업주에게 직접고용 간주(2년 초과 또는 허용 업종 외)** `[모델 지식 — 검증 필요]`
- 형사 처벌(파견법 위반) 및 행정제재 병행 가능

### 8.3 기간제·단시간 근로자 차별 금지

대상 관계가 근로자로 분류된 경우, 추가로:
- **기간제법 §8:** 기간제 근로자에 대한 합리적 이유 없는 차별 금지
- **비교 대상 근로자:** 동종·유사 업무 수행 무기계약 근로자 비교
- **2년 초과 고용:** 기간제법 §4 — 무기계약 전환 간주 `[모델 지식 — 검증 필요]`
- 단시간 근로자: 시간비례 원칙, 초단시간(주 15시간 미만) 특칙 확인

## 9. 소급 노출 추정 (보수적)

> ⚠️ 수치는 사용자가 제공한 **보수·기간 데이터 기반으로만** 산출한다.
> 데이터가 없으면 공식만 제시하고 수치를 채워 넣지 않는다.

### 필요 입력 데이터 (사용자 제공)

| 항목 | 값 |
|---|---|
| 월 보수(세전) | |
| 계약 시작일 | |
| 현재/종료일 | |
| 사업장 상시 근로자 수 | |
| 주 근무시간(추정) | |

---

### 9.1 퇴직금 (근로자퇴직급여보장법)

**적용:** 계속근로기간 1년 이상, 주 15시간 이상

```
퇴직금 = 평균임금 × 30일 × (계속근로일수 / 365)
평균임금 = 퇴직 전 3개월 총 임금 / 3개월 달력일수
```

> [데이터 입력 시 여기에 계산값 기재. 데이터 없으면: "보수·근속기간 입력 후 산출 가능"]

---

### 9.2 4대보험 미납분 소급 (사업장 분담 추정)

**보수적 추정 기준** `[모델 지식 — 검증 필요]` — 요율은 고용노동부 현행 고시 확인:

| 보험 | 사업장 분담 요율(추정) | 근로자 분담 요율(추정) |
|---|---|---|
| 국민연금 | 4.5% | 4.5% |
| 건강보험 | ~3.5% (변동) | ~3.5% |
| 고용보험 | ~0.9%+ (규모별) | 0.9% |
| 산재보험 | 업종별 (0.7%~) | — |

```
사업장 소급분 ≈ 월 보수 × 사업장 분담 요율 합계 × 소급 월수
근로자 분담 추정분 ≈ 월 보수 × 근로자 분담 요율 합계 × 소급 월수
```

> 실제 요율은 고용노동부·국민연금공단·건강보험공단 현행 고시 확인 필수 `[모델 지식 — 검증 필요]`.
> [데이터 입력 시 여기에 계산값 기재]

---

### 9.3 연차미사용수당 소급

**적용:** 5인 이상 사업장, 계속근로 1년 이상 시 연 15일 기준 (근기법 §60) `[모델 지식 — 검증 필요]`

```
연차미사용수당 = 통상임금 × 미사용 연차일수
연차 발생 기준: 1년 개근 → 15일, 이후 2년마다 1일 가산(최대 25일)
```

> [데이터 입력 시 여기에 계산값 기재]

---

### 9.4 가산수당 소급 (5인 이상 사업장)

**적용:** 근기법 §56, 5인 이상 사업장

```
연장근로 가산: 통상임금의 50% 가산
야간근로(오후 10시 ~ 오전 6시) 가산: 통상임금의 50% 가산
휴일근로 가산: 8시간 이내 50%, 8시간 초과 100%
```

> [주 실제 근무시간·야간·휴일 근무 데이터 입력 시 산출 가능]

---

## 11. 검토자 메모

> ⚠️ **이 분석은 검토자(변호사 또는 공인노무사)의 판단을 대체하지 않는다.**

- 종속노동성 판단은 **사실관계 전체를 종합**한 법적 판단이며, 이 스킬은 체크리스트 기반 위험 스크리닝을 제공한다
- 법망 MCP 미연결 시: 모든 법령 인용·판례 참조는 `[모델 지식 — 검증 필요]` — 법망 현행 DB와 대조 전 의존 금지
- 플랫폼 종사자 보호법 관련 내용은 입법 동향이 자주 변경된다 `[모델 지식 — 검증 필요]` — 현행 시행 여부 및 적용 범위는 반드시 고용노동부 공식 자료로 확인
- 소급 노출 수치는 확정액이 아닌 **보수적 추정**이며, 실제 협상·정산 시 전문가 검토 필요
- 불법파견으로 판단될 경우 형사 책임이 병행될 수 있으므로 즉시 법무 에스컬레이션

## 12. 다음 단계 의사결정 트리

아래 4가지 경로 중 선택:

```
근로자성 분석 결과
├── 🔴 고위험 (근로자 방향 요소 다수)
│   ├── [A] 계약구조 재설계
│   │       현재 계약을 독립 사업자 요건에 맞게 재설계 (업무지시 제거, 성과 기반 보수, 도구 자가 제공 등)
│   │       → 재설계 후 `/employment-legal:worker-classification` 재평가 권고
│   ├── [B] 소급 정리 계획
│   │       오분류를 인정하고 4대보험 소급 납부, 퇴직금 지급, 근로계약 전환
│   │       → 노무법인·세무사 협력으로 소급 정리 로드맵 수립
│   └── [C] 에스컬레이션 (법무 또는 노무법인)
│           고용노동부 조사·노동위원회 구제신청 대응, 소송 방어 전략
│           → 즉시 담당 변호사·공인노무사에게 전달
│
├── 🟡 중위험 (양방향 혼재)
│   ├── 추가 사실확인 목록 작성 후 재평가
│   └── [D] 현상유지 + 모니터링
│           계약 구조 변경 없이 위험 요소 목록화하고 정기 재검토 (분기 1회 권고)
│           → 실무 프로파일에 플래그 기록
│
└── 🟢 저위험 (비근로자 방향 요소 다수)
    └── [D] 현상유지 + 모니터링
            현재 구조 유지하되, 사실관계 변경(근무시간 증가, 전속성 심화 등) 시 재평가
```
