# /labor-committee-procedure

미국 원본 대응 없는 KR-NEW 스킬. 노동위원회(노동위)는 한국 고유의 준사법적 행정기관 — 부당해고·
부당노동행위 구제의 1차 forum. 이 스킬은 절차 유형 판정부터 기한 관리, 판정례 데이터 갭 대응,
관련 경로(진정·민사소송) 병행 검토까지 안내한다.

---

## 1. 매터 컨텍스트

1. `~/.claude/plugins/config/claude-for-legal-kr/employment-legal/CLAUDE.md` 읽기 — 상시 근로자
   수·사업장·노조 여부·에스컬레이션 체인·플레이북 확인.
2. 활성 매터가 있으면 `matter.md` 읽기 — 사건 당사자·해고일·판정서 등 매터-특화 사실 확인.
3. 국가법령정보 MCP 연결 여부 확인. 미연결이면 검토자 메모에 기록: "국가법령정보 미연결 — 조문 인용은 모델 지식,
   의존 전 검증 필요."

---

## 2. 목적지 점검

산출물 생성 전 수신자·배포 범위 확인:

- **내부 법무·노무 검토용:** 비밀유지 헤더 적용 (CLAUDE.md `## 산출물` 기준).
- **외부 노동위 제출용:** 헤더 제거, 당사자 기재 확인.
- **상대방(사용자·근로자)에게 직접 전달:** 목적지 점검 엄격 적용 — 비밀유지 서클 외부 여부 확인.
- **직장 내 괴롭힘·성희롱 관련 사건:** 비밀누설 금지 의무(근기법 §76-3·남녀고용평등법 §14) 특히 강함.

---

## 3. 목적

이 스킬의 역할:

1. 해당 노동위 절차 유형 판정 (부당해고 / 부당노동행위 / 재심 / 행정소송 / 기한 확인)
2. 적용 가능성 게이트 체크 (5인 미만 사업장, 근로자성, 기한)
3. 절차 단계별 실무 안내 (신청서 요소, 제출처, 타임라인, 구제 내용 선택)
4. 기한 계산 및 관리 (구제신청 3개월, 재심 10일, 행정소송 15일/90일)
5. 노동위 판정례 데이터 갭 대응 — 이 스킬의 1급 관심사

## 5. 절차 유형 판정

아래 5개 경로 중 해당하는 것 확인. $ARGUMENTS 또는 사용자 제공 사실로 판정. 불명확하면 질문.

| 경로 | 해당 상황 | 이동 섹션 |
|---|---|---|
| **A — 부당해고 구제신청** | 해고를 받은 근로자 또는 그 대리인 | §7, §8 |
| **B — 부당노동행위 구제신청** | 노조 활동·단체교섭 관련 불이익 | §8 (부당노동행위), §9 |
| **C — 중노위 재심** | 이미 지노위 판정 받음, 재심 검토 | §9 (재심) |

## 6. 적용 가능성 체크 — 부당해고 구제신청 (경로 A)

본 섹션은 경로 A(부당해고)에만 적용. 경로 B는 §9로.

### 6-1. 상시 근로자 5인 이상 여부

> **게이트.** 근기법 §28(부당해고 구제신청)은 §11의 적용범위 조항을 통해 **상시 근로자 5인 이상
> 사업장**에만 적용된다. `[모델 지식 — 검증 필요]` `[상시 근로자 수 산정 — 검증 필요]`

산정 방법: 국가법령정보에서 근기법 §11 및 시행령 산정기준 조회.

```
mcp__korean-law__get_law_text "근로기준법 제11조"
mcp__korean-law__legal_analysis {"mode": "verify_citations", "citation": "근로기준법 시행령 제7조의2"}
```

**5인 미만이면:**
- §28 구제신청 불가 — 노동위 경로 종료
- 대안 경로:
  - **고용노동부 진정** (근기법 위반 사항에 대한 근로감독 → 형사책임 추궁)
  - **민사소송** (해고무효확인 + 임금청구 — 법원, 기한 없음)
  - 퇴직금·임금체불은 5인 미만에도 적용
- 사용자에게 경로 전환 안내 후 스킬 종료 또는 민사·진정 경로로 핸드오프

**산정 불명확하면:** `[상시 근로자 수 산정 — 검증 필요]`로 플래그하고 계속 진행하되 5인 미만인 경우
구제신청 불가 가능성을 검토자 메모에 명시.

### 6-2. 근로자성 확인

- **도급·특수형태근로종사자(특고)·프리랜서:** 원칙적으로 §28 구제신청 불가 `[검토]`
- 근로자성이 불명확하면: `/employment-legal:worker-classification` 핸드오프
  - 핸드오프 전 현재 사실 요약 전달
  - 근로자성 판정 결과 가져온 후 이 스킬 재개

### 6-3. 해고일로부터 3개월 기한 확인

> **엄격한 기한.** 구제신청 기한 도과 → 각하. 연장 없음. `[모델 지식 — 검증 필요]`
>
> `[검토]` — 구두 해고와 서면통지 사이에 시간적 괴리가 있는 경우 기산점이 다투어진다.
> 이 쟁점은 **양방향**으로 작동한다 — 어느 쪽이 유리한지는 사실관계에 따라 다르므로 단정 금지:
>
> **① 기한이 앞당겨지는 방향 (의뢰인에게 불리할 수 있음)**
> 구두 해고 통보가 서면통지보다 먼저 이루어진 경우, 기산점이 구두 통보일로 앞당겨질 수 있다.
> 서면통지서 날짜를 기준으로 계산했더니 기한 내이더라도, 그 이전 구두 해고 시점을
> 기산점으로 보면 기한이 이미 도과한 것으로 판정될 위험이 있다. `[노동위 판정 — 직접 검증 필요]`
>
> **② 기한이 뒤로 밀리는 방향 (의뢰인에게 유리할 수 있음)**
> 서면통지서가 해고 의사표시의 **첫 번째** 공식 통지이고 그 이전에 구두 해고가 없었다면,
> 기산점은 서면통지서 교부일로 확정된다. 겉보기에 기한이 도과한 것처럼 보이는 사안에서도
> "서면통지가 구두 해고보다 실제로 늦게 교부되었는가"를 사실관계 기준으로 재확인할 필요가 있다.
> `[노동위 판정 — 직접 검증 필요]`
>
> **실무 적용:** 해고일이 불명확하거나 구두 통보와 서면통지 날짜가 다른 경우, 양방향 시나리오를
> 모두 검토하고 `[검토]`로 플래그. 어느 쪽이 적용되는지는 변호사·노무사 판단 사항.

**날짜 계산 (사용자가 날짜를 제공한 경우):**

```
해고일:         [사용자 제공]
구제신청 마감:  해고일 + 3개월
오늘:          [오늘 날짜]
남은 기간:     [계산]
```

날짜를 제공하지 않으면: 공식만 제시, 계산하지 않음.

**기한 임박(30일 이내):** 🔴 차단 플래그 — 즉시 신청 또는 변호사·노무사 긴급 자문 에스컬레이션.

## 7. 부당해고 구제신청 절차 — 상세 (경로 A)

### 7-1. 신청서 작성 요소

부당해고 구제신청서 필수 기재사항 (`[모델 지식 — 검증 필요]`, 노동위 서식 대조 필요):

| 항목 | 내용 |
|---|---|
| 신청인 | 성명, 주소, 연락처 |
| 피신청인 | 사용자 명칭, 주소, 대표자 |
| 해고 일자 | 정확한 날짜 — 기산점 기준 |
| 해고 사유 | 사용자가 통지한 사유 (§27 서면통지 내용) |
| 구제 내용 | 원직복직 / 금전보상명령 (아래 §7-3 선택 참고) |
| 청구 취지 | "부당해고임을 확인하고 원직에 복직시켜 달라" 또는 금전보상 |

### 7-2. 제출처

- **제출:** 해당 사업장 관할 **지방노동위원회(지노위)**
- 관할: 주된 사업장 소재지 기준 `[검토]`
- 국가법령정보에서 관할 지노위 확인 가능:
  ```
  mcp__korean-law__search_decisions {"query": "지방노동위원회 관할"}
  ```

### 7-3. 구제 내용 선택 — 원직복직 vs 금전보상명령

> 근기법 §33 금전보상명령 `[모델 지식 — 검증 필요]`

| 구분 | 원직복직 | 금전보상명령 (§33) |
|---|---|---|
| **내용** | 해고 전 동일 직위·업무 복직 + 해고기간 임금상당액 지급 | 복직 대신 금전 지급 |
| **신청인** | 근로자 신청 가능 | 근로자 신청 가능 (2021년 개정 후 일정 요건에서 사용자도 신청 가능 `[검토]`) |
| **인정 기준** | — | 사회통념상 근로관계 계속이 어려운 경우 `[검토]` |
| **장점** | 지위·경력 회복 | 복직 없이 확정적 보상 |
| **단점** | 사용자 저항·현실적 복귀 어려움 가능 | 금액이 충분하지 않을 수 있음 |

`[검토]` — 금전보상 금액 기준, 사용자 신청 가능 범위는 판정례·판례 대조 필요. `[노동위 판정 — 직접 검증 필요]`

### 7-4. 절차 단계 타임라인

```
[신청서 제출 — 지노위]
        ↓
[접수 + 사건번호 부여]
        ↓
[사건담당 조사관 배정]
        ↓
[사용자(피신청인) 답변서 제출]  ← 보통 2~3주
        ↓
[조사관 조사]  ← 당사자 면담·문서 검토
        ↓
[심문회의]  ← 당사자 출석, 진술·반박
        ↓
[판 정]  ← 목표 60일, 실제로 더 걸리기도
        ↓
[판정서 수령]
```

> `[모델 지식 — 검증 필요]` 타임라인은 실제 사건별로 다름. 노동위 처리 현황은 노동위 홈페이지
> 또는 담당 조사관에게 확인.

## 8. 부당노동행위 구제신청 절차 — 상세 (경로 B)

> 근거: 노동조합 및 노동관계조정법(노조법) §81, §82 `[모델 지식 — 검증 필요]`

### 8-1. 부당노동행위 유형 체크

해당 행위 유형 확인 (복수 해당 가능):

| 유형 | 내용 | 노조법 |
|---|---|---|
| **불이익취급** | 노조 활동·쟁의행위 참가를 이유로 한 해고·불이익 | §81①1호 |
| **황견계약** | 노조 불가입·탈퇴·특정 노조 가입 조건 채용·계속고용 | §81①2호 |
| **단체교섭 거부** | 정당한 이유 없는 단체교섭 거부·해태 | §81①3호 |
| **지배·개입** | 노조 조직·운영에 지배 또는 개입 | §81①4호 |
| **기금원조** | 노조 운영비 원조 (단, 일부 예외) | §81①5호 |

유형 불명확하면 `[검토]` 플래그.

### 8-2. 신청인

- **근로자** (개인) — 불이익취급·황견계약
- **노동조합** — 단교거부·지배개입·기금원조 (노조가 단체로 신청)
- 사실에 따라 둘 다 신청 가능한 경우도 있음 `[검토]`

### 8-3. 기한 — 3개월 (계속행위 주의)

> **엄격한 기한.** `[모델 지식 — 검증 필요]`

- **기산점:** 행위가 있은 날
- **계속행위:** 행위가 계속되고 있으면 **종료일**부터 3개월 `[검토]`
  - 예: 단교 거부가 지속되는 경우 → 마지막 거부 시점 기준
  - 이 기산점 판단은 사안별 — `[검토]` 플래그 필수

**날짜 계산 (사용자가 날짜를 제공한 경우):**

```
행위일(또는 계속행위 종료일):  [사용자 제공]
구제신청 마감:                 행위일 + 3개월
오늘:                         [오늘 날짜]
남은 기간:                    [계산]
```

### 8-4. 절차

부당해고와 동일한 지노위 → 중노위 → 행정소송 구조 (§9 참고).  
구제 내용: 원상회복 (복직·단교 재개 등) 명령. 금전보상명령은 부당해고 전용 — 부당노동행위에는
적용 다름 `[검토]`.

## 9. 중노위 재심 및 행정소송 (경로 C·D)

### 9-1. 중노위 재심신청 (경로 C)

> 근거: 근기법 §31 `[모델 지식 — 검증 필요]`

**기한: 지노위 판정서를 받은 날로부터 10일 — 엄격, 연장 없음**

```
지노위 판정서 수령일:    [사용자 제공]

오늘:                  [오늘 날짜]
남은 기간:             [계산]
```

> ⚠️ **10일은 매우 짧다.** 판정서 수령 즉시 변호사·노무사 검토 에스컬레이션 필요.

**중노위 재심 포인트:**

- 새로운 증거 제출 가능 여부 `[검토]`
- 중노위는 지노위 기록 전부 심리 + 새 심문 가능

### 9-2. 행정소송 (경로 D)

> 근거: 근기법 §32, 행정소송법 `[모델 지식 — 검증 필요]`

**⚠️ 기한 복잡성 — `[검토]` 필수**

| 근거 | 기한 | 비고 |
|---|---|---|
| 근기법 §32 | 중노위 판정서 받은 날로부터 **15일** | `[검토]` — 실무·판례 대조 필요 |

> `[검토]` 두 기한의 관계 및 우선 적용 기준은 국가법령정보 조문 대조 + 변호사 검토 필수.
> 단정하지 않음 — 짧은 기한(15일)을 보수적 기준으로 사용하는 것이 안전하나, 이 판단 자체가 `[검토]`.

**피고:** 중앙노동위원회 위원장 `[모델 지식 — 검증 필요]`

**관할 법원:**
- 중노위 사건: 서울행정법원 `[모델 지식 — 검증 필요]` `[검토]`
- 구체 관할은 국가법령정보 또는 변호사 확인

**날짜 계산 (사용자가 날짜를 제공한 경우):**

```
중노위 판정서 수령일:   [사용자 제공]

행정소송 마감 (90일):  수령일 + 90일  [검토]
```

### 9-3. 이행강제금

> 근기법 §33의2 이하 `[모델 지식 — 검증 필요]`

- **발동:** 노동위의 복직명령 확정 후 **30일** 이내 사용자가 이행하지 않을 때

## 12. 고용노동부 진정과의 관계

### 12-1. 경로 비교

| 경로 | 목적 | 주체 | 결과 |
|---|---|---|---|
| **노동위 구제신청** | 근로자 구제 (복직·임금) | 근로자/노조 신청 | 구제명령 (이행강제금 가능) |
| **고용노동부 진정** | 사용자 형사책임 추궁 | 근로자 진정 → 근로감독관 수사 | 과태료·고발·시정명령 |
| **민사소송** | 해고무효확인 + 임금청구 | 근로자 소제기 | 법원 판결 |

### 12-2. 병행 가능성

세 경로는 **동시 진행 가능**. 상호 배타적이지 않음 `[모델 지식 — 검증 필요]`.

- 노동위 구제신청 + 진정: 가능 — 목적·결과가 다름
- 노동위 구제신청 + 민사소송: 가능 — 단, 판정 결과와 민사 결과의 기판력 관계 `[검토]`
- 진정 + 민사소송: 가능

### 12-3. 전략적 고려

- 구제신청: **빠름** (목표 60일), **비용 저렴** (인지대 없음), 복직·임금상당액 구제
- 민사소송: 법원 기판력 있는 판결, 구제신청 결과에 구속 안 됨, **시간·비용 큼**
- 진정: 사용자 처벌 목적, 근로자 직접 구제 아님

`[검토]` — 어느 경로를 우선할지는 사안별 전략 판단. 변호사·노무사 자문 권장.
