## 목적지 점검

취업규칙 변경 관련 산출물은 배포 전 목적지 점검 필수. 동의 절차 문서(서면 동의서·회의 기록)는
특히 민감 — 노동청 감독·노동위 절차에서 자료제출 요구 대상이 될 수 있다. 사용자가 목적지를
명시했으면 비밀유지 서클 내부인지 확인한다. 서클 외부(전사 공지·직원 대상 통지·외부)이면 내용과
형식을 달리한다 — 플래그하고 (a) 법무용 버전, (b) 직원 공지용 버전 중 무엇을 줄지 묻는다.
실무 수준 CLAUDE.md `## 공유 가드레일 → 목적지 점검`이 정본.

---

## 목적

**기존 취업규칙을 변경**할 때 발생하는 법적 의무를 점검하고 절차를 안내한다.

- **불이익변경 여부 판단** — 변경 내용이 근로자에게 불이익한지 분석 (`[검토]` 필수)
- **동의 절차 안내** — 불이익변경 확정 시 노동조합 또는 근로자 과반수 동의 절차
- **변경 전후 비교** (`--compare`) — 변경 전 조항과 변경 후 조항을 대조해 불이익 항목 식별
- **동의 절차 문서 초안** (`--consent`) — 서면 동의서·근로자 회의 의사록 양식 초안

한국에는 미국식 "state supplement(주별 보충)" 개념이 없다 — 근로기준법은 전국 단일 관할.
지역별 추가 요건 점검은 불필요하다.

산출물은 법무·HR·공인노무사의 검토 기초 — 불이익 여부를 AI가 단정하지 않는다. 판단 지점은 모두 `[검토]`.

## 변경 내용 수집

분석 시작 전 변경 내용을 구체적으로 수집한다. 제공되지 않았으면 묻는다:

> 1. **무엇이 바뀌나요?** — 변경 전 조항 텍스트와 변경 후 조항 텍스트를 가능하면 같이 붙여주세요.
> 2. **변경 이유가 무엇인가요?** — 경영상 필요·법령 개정 반영·노사 합의 등
> 3. **노동조합이 있나요?** — 과반수 노동조합 유무에 따라 동의 절차가 달라집니다.
> 4. **상시 근로자 수는 몇 명인가요?** — 10인 이상이면 변경 신고 의무가 있습니다.

`--compare` 모드: 사용자가 변경 전·후 조항을 제공하면 섹션별 대조표를 먼저 작성한다.

## 불이익변경 여부 판단

`[검토]` — 불이익 여부는 전체 취업규칙을 **종합·비교**해 판단한다. AI가 단정하지 않는다.
`[모델 지식 — 검증 필요]`

**판단 원칙:**
- 일부 조항 개선·일부 조항 불이익이 **혼재**하는 경우: 전체를 종합해 불이익 여부를 판단한다. `[검토]`
- 개별 근로자마다 영향이 다를 수 있다 — 특정 집단에만 불이익이 집중되는 구조라면 추가 분석 필요. `[검토]`
- 판례는 "사회통념에 비추어 볼 때 전체적으로 근로자에게 불이익"한지를 기준으로 한다.
  `[모델 지식 — 검증 필요]`

### 변경 유형별 불이익 판단 기준

| 변경 내용 | 불이익 여부 | 심각도 | 비고 |
|---|---|---|---|
| 임금 삭감·기본급 인하 | 명백히 불이익 | 🔴 | 동의 필수 |

| 소정근로시간 연장 | 명백히 불이익 | 🔴 | 동의 필수 |
| 징계 사유 추가 | 불이익 가능 | 🟠 `[검토]` | 사유의 내용·범위에 따라 판단 |
| 징계 양형 강화 (경고→감봉, 감봉→해고) | 불이익 가능 | 🟠 `[검토]` | 동의 필요 가능성 高 |
| 복리후생 축소 (식대·교통비 감액) | 불이익 가능 | 🟠 `[검토]` | 지급 기준·금액에 따라 판단 |
| 휴가 일수 감소 | 불이익 가능 | 🟠 `[검토]` | 법정 기준 미달이면 위법 별도 |
| 정년 단축 | 명백히 불이익 | 🔴 | 동의 필수 |
| 경쟁금지·비밀유지 조항 신설·강화 | 불이익 가능 | 🟠 `[검토]` | 범위·기간·보상 종합 판단 |
| 일부 개선·일부 불이익 혼재 | 종합 판단 필요 | `[검토]` | 전체 비교 + 변호사·노무사 판단 |
| 근로자에게 유리한 변경 | 유리 | 🟢 | 의견청취로 충분 |

**각 항목의 불이익 여부 판단 결과를 명시한다:** "[변경 항목]은 [불이익 / 유리 / 혼재] 가능성이
있습니다. 이유: [1-2줄 사실 요약]. `[검토]`"

## 동의 절차 안내

### 불이익변경이 확정·가능성 있는 경우

`[모델 지식 — 검증 필요]` — 동의 방법·절차 요건은 법망 §94 현행 조문 대조 필수.

#### 케이스 A: 과반수 노동조합이 있는 경우

> **동의 주체:** 근로자 과반수로 조직된 노동조합.
>
> **절차:**
> 1. 변경 내용·이유를 노동조합에 서면 통보
> 2. 노동조합과 의견 교환·협의
> 3. **노동조합의 서면 동의** 획득
> 4. 서면 동의서 보관 (노동청 감독·분쟁 대비)
> 5. 변경 취업규칙 고용노동부 신고 (10인 이상, §93)
> 6. 근로자에게 변경 내용 주지 (§14)
>
> `[검토]` — 노동조합이 "과반수"인지 확인 필요. 과반수 미달이면 케이스 B 적용 검토.

#### 케이스 B: 과반수 노동조합이 없는 경우

> **동의 주체:** 근로자 과반수의 서면 동의.
>
> **절차:**
> 1. **근로자 회의 개최** — 전체 근로자 또는 사업장별 근로자를 대상으로 회의 소집
> 2. 변경 내용·이유 설명 및 질의응답
> 3. 자유로운 의사에 따른 **과반수 서면 동의** 취득
>    - 서명부 방식 또는 서면 동의서 방식
>    - 동의 의사는 자유롭게 형성되어야 함 — 사용자가 동의를 강요하거나 불이익을 예고하면 동의 효력 부정 위험 `[검토]`
> 4. 동의서·회의 기록 보관 (노동청 감독·분쟁 대비)
> 5. 변경 취업규칙 고용노동부 신고 (10인 이상)
> 6. 근로자에게 변경 내용 주지 (§14)

**단계별 체크리스트:**

| # | 단계 | 완료 여부 | 비고 |
|---|---|---|---|
| C-1 | 노동조합 유무 확인 및 과반수 여부 판단 | ☐ | |
| C-2 | 변경 내용·이유 서면 통보 또는 공지 | ☐ | |
| C-3 | 회의 개최 (케이스 B) 또는 노동조합 협의 (케이스 A) | ☐ | |
| C-4 | 과반수 서면 동의 또는 노동조합 서면 동의 획득 | ☐ | |
| C-5 | 동의서·회의 기록 보관 | ☐ | 분쟁 시 입증 책임 |
| C-6 | 변경 취업규칙 고용노동부 신고 (10인 이상) | ☐ | |
| C-7 | 근로자 주지 (게시 또는 교부) | ☐ | §14 의무 |

### 동의 없이 불이익변경 강행 시

해당 변경 조항은 **무효** — 변경 전 구 취업규칙이 해당 부분에 계속 적용된다.
`[모델 지식 — 검증 필요]`

> 동의 없이 강행하면 법적 효력이 없는 것은 물론, 노사 신뢰 훼손과 노동청 진정·분쟁의
> 기폭제가 될 수 있습니다. 노사 마찰 위험이 법적 위험과 별개로 높습니다. 🔴

## 사회통념상 합리성 예외

대법원은 극히 예외적으로, 근로자 동의 없이도 불이익변경이 유효할 수 있다고 인정해왔다.
`[모델 지식 — 검증 필요]`

**단, 이 예외는 엄격하고 좁다. AI가 단정하지 않는다 — 반드시 `[검토]`.**

예외 인정 가능 고려요소 (판례 기준 — 법망·판례 직접 확인 필요):
- 변경의 불이익 정도
- 사용자 측의 변경 필요성의 내용·정도
- 변경 후 내용의 상당성
- 대상 조치(보상·유예기간 등) 유무
- 노동조합 등과의 교섭 경위
- 다른 근로조건의 개선 상황

> ⚠️ **사회통념상 합리성 예외 사용 주의:**
> 이 예외를 전제로 동의 절차를 생략하면 나중에 법원이 예외를 부정했을 때 변경 전체가
> 무효로 돌아올 위험이 있습니다. `[검토]` 변호사·노무사 자문 후 결정을 강력히 권고합니다.
> `[모델 지식 — 검증 필요 / 판례 직접 확인 필요]`

## 유리한 변경 절차

변경이 근로자에게 **유리**한 경우 동의 없이 **의견청취**만으로 충분하다.
`[모델 지식 — 검증 필요]`

**절차:**
1. 근로자 과반수(또는 노동조합)에게 변경 내용을 알리고 의견을 듣는다
2. 의견청취 기록 보관
3. 변경 취업규칙 고용노동부 신고 (10인 이상)
4. 근로자 주지 (§14)

**주의:** "유리한 변경"이라도 일부 근로자에게 불리한 측면이 있으면 해당 부분은 불이익변경으로
처리해야 한다. `[검토]`

## 변경 후 고지·신고

### §14 주지 의무 `[모델 지식 — 검증 필요]`

변경된 취업규칙은 **근로자에게 주지**해야 한다.

주지 방법:
- 사업장 내 게시 또는 비치
- 근로자에게 교부
- 사내 인트라넷·이메일 등 접근 가능한 방법 `[검토 — 전자적 방법의 충분성은 사안별 판단]`

주지하지 않으면: 변경된 취업규칙 내용을 근로자에게 효력 주장하기 어려울 수 있다. `[검토]`

### 고용노동부 변경 신고 (10인 이상 사업장)

`[모델 지식 — 검증 필요]` — 현행 절차·서류는 고용노동부 민원마당 또는 지방관서에서 확인.

**신고 방법:**
- 온라인: 고용노동부 민원마당(minwon.moel.go.kr)
- 방문·우편: 사업장 소재지 관할 지방고용노동관서

**제출 서류 (확인 필요):**
- [ ] 변경된 취업규칙
- [ ] 근로자 과반수 동의(또는 의견청취) 증빙 서류
- [ ] 기타 관할 관서 요구 서류 `[검토 — 현행 서식 지방관서 확인]`

**신고 기한:** `[모델 지식 — 검증 필요]` — 변경 후 신고 기한은 법망·지방관서 확인 필수.

## 개별 근로계약 vs 변경된 취업규칙 (§97)

변경된 취업규칙이 **기존 개별 근로계약보다 불리**한 경우:
→ 해당 근로자에 대해서는 개별 근로계약이 우선 적용된다. `[모델 지식 — 검증 필요]`

> 즉, 취업규칙을 불이익하게 변경하더라도, 기존 근로계약이 더 유리한 조건을 정했다면
> 그 근로자에게는 변경 효력이 미치지 않을 수 있습니다. `[검토]`
> 개별 근로계약서를 확인해 §97 충돌 여부를 함께 점검하세요.

  진행하면 동의 자체가 무효로 판단될 위험 — 절차 안내에 이 경고를 포함.

## 다음 단계 의사결정 트리

**제 체크리스트에는 없지만 한 가지 짚어드리고 싶은 것:** 취업규칙 변경 후 개별 근로계약서와
충돌 여부(§97)를 확인하지 않으면 특정 근로자에게는 변경 효력이 미치지 않는 경우가 생깁니다.
특히 연봉계약서·개별 합의서로 별도 조건을 정한 근로자가 있다면 계약서를 함께 검토하면
추가 분쟁을 예방할 수 있습니다.

> **다음 단계? 하나 골라주세요. 살을 붙이겠습니다:**
>
> 1. **동의 절차 메모 초안** — 근로자 과반수 서면 동의 또는 노동조합 동의를 위한
>    내부 메모 초안(변경 이유·내용 설명·동의 요청). 직원용 공지 버전도 가능합니다.
>
> 2. **근로자 회의 진행 안내** — 회의 소집 통보문·진행 순서·동의서 양식·의사록 양식 초안.
>    (케이스 B — 과반수 노동조합 없는 사업장)
>
> 3. **고용노동부 신고 서류 체크리스트** — 변경 취업규칙 신고에 필요한 서류 목록·
>    제출 순서·기한 정리. 10인 이상 사업장이면 신고 기한 함께 확인.
>
> 4. **에스컬레이션** — [실무 프로파일의 에스컬레이션 대상]에게 보낼 짧은 메모
>    (불이익 여부 요약 + 동의 절차 필요 여부 + 필요한 결정 사항).
>
> 5. **관찰·대기** — 트래커에 등재. 동의 절차 진행 상황·기한·다음 점검 시점 메모.
>
> 6. **다른 것** — 이 결과로 무엇을 하고 싶은지 알려주세요.
