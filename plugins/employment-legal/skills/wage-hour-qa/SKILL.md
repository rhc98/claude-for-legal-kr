## GATE: 상시 근로자 수 확인

**모든 후속 분석의 게이트다.** 분석 시작 전 상시 근로자 수를 확인한다.

상시 근로자 수를 모르면 묻는다:
> "분석의 전제가 되는 **상시 근로자 수**를 알려주세요. 5인 미만 사업장은 근기법 §56
> 가산수당(연장·야간·휴일)이 적용되지 않아 분석 결과가 크게 달라집니다."

| 임계점 | 주요 적용·제외 (현행 시행령 대조 필요) |
|---|---|
| **5인 미만** | §56 가산수당 ✗, §23 해고제한·§28 부당해고구제 ✗, §60 연차 ✗, 휴업수당 등 다수 제외 |
| **5인 이상** | 가산수당·연차·해고제한 적용 |

## §C — 가산수당 산정 워크시트

**적용 전제:** 5인 이상 사업장. 5인 미만이면 §56 가산수당 미적용 — GATE에서 이미 확인.

### 가산율 (현행 §56 기준 — 국가법령정보 조회로 확인)

| 구분 | 가산율 | 기준 시간 |
|---|---|---|
| 연장근로 | +50% | 소정근로 초과분 |
| 야간근로 | +50% | 22:00 ~ 06:00 실제 근로 시간 |
| 휴일근로 (8시간 이내) | +50% | 주휴일·법정공휴일 |
| 휴일근로 (8시간 초과) | +100% | 8시간 초과분 |

`[모델 지식 — 검증 필요]` — 국가법령정보 현행 §56 조문 대조 필수.

> **중복 가산:** 야간근로가 연장근로와 겹치면 두 가산이 중복 적용된다. 휴일근로가 야간과
> 겹치면 마찬가지. 사실관계 확인 필수.

### 산정 공식 (사용자 데이터 필요)

**통상임금 시급 산출:**
```
통상임금 월 지급액 ÷ (소정근로시간수 + 주휴시간수) = 통상임금 시급
```
- 주 40시간 사업장: 월 환산시간 = (40 + 8) × 52 / 12 ≒ 209시간 `[검토 — 취업규칙·근로계약 확인]`

**가산수당 산정:**
```
연장수당 = 통상임금 시급 × 연장근로시간 × 1.5
야간수당 = 통상임금 시급 × 야간근로시간 × 0.5  (연장과 중복 시 추가)
휴일수당 = 통상임금 시급 × 휴일근로시간 × 1.5 (8h 이내) 또는 × 2.0 (초과분)
```

**미지급 추산 (사용자 데이터 필요):**

| 기간 | 연장시간 | 야간시간 | 휴일시간 | 산출 가산수당 | 실지급액 | 미지급 추산 |
|---|---|---|---|---|---|---|
| [월] | | | | | | |

## §E — 포괄임금제 유효성 체크

> **§E 진입 전 필수 선행 단계 — §A 통상임금 판단**
>
> 계약에 포괄임금 수당 항목(예: "연장근로 포함 수당", "야간 포함 고정 OT", "시간외 일괄 지급액" 등)이
> 있으면 **§A 통상임금 판단을 먼저 실행**한다. 해당 수당이 통상임금에 산입되는지 여부가
> 가산수당(§C) 산정의 시급 기초를 바꾸기 때문이다.
>
> - 포괄임금 수당 → 통상임금 **산입 시:** 통상임금 시급이 높아짐 → 미지급 가산수당 추산 증가
> - 포괄임금 수당 → 가산수당 성격 분리 시: 통상임금 기초에서 제외 → 시급 계산 달라짐
>
> §A 판단 없이 §E·§C만 실행하면 미지급 추산이 과소 또는 과대가 된다. 포괄임금 수당 항목이
> 있을 때마다 §A → §C → §E 순서를 지킨다.

**핵심 원칙:** 포괄임금제는 근로시간 산정이 실제로 어려운 경우에 한해 제한적으로 유효하다.
법원은 유효성을 엄격하게 심사하는 추세다. `[검토]` 항상 필요.

**유효 가능 요건 (판례 기준 — 국가법령정보 조회 확인):**
1. 근로시간 산정이 어렵거나 곤란한 업무인가?
   - 감시·단속적 근로 (경비·시설관리 등)
   - 재량근로 대상 업무
   - 그 외 사실상 근로시간 계산이 어려운 경우 `[검토]`
2. 근로계약·취업규칙에 포괄임금 합의가 명시되어 있는가?
3. 포괄임금에 포함된 가산수당이 실제 법정 산정액보다 적지 않은가?

**무효 위험 지표 (아래 해당 시 `[검토]` 플래그):**
- 사무직·통상 사무 업무에 포괄임금 적용
- 실근로시간과 포괄임금 포함 시간의 괴리가 큰 경우
- 포괄임금액이 법정 가산수당 합계에 미치지 못하는 경우
- 근로계약서에 포괄임금 합의가 불명확한 경우

**무효 선언 시 효과:**
- 실근로시간에 따라 가산수당 전액 재산정·지급 의무
- 소멸시효(3년) 범위 내 미지급분 소급 청구 위험

**검토 결과 표시:** 항상 `[검토]` — 포괄임금 유효성은 법원이 사실관계 전체를 보고 판단하는
사항이다. AI가 유효/무효를 단정하지 않는다.

## §F — 최저임금 준수 체크

> **중요:** 최저임금은 매년 고시된다. 금액은 절대 하드코딩하지 않는다.
> 반드시 `references/employment-currency-watch.md` 또는 최저임금위원회 공식 사이트에서
> 해당 연도 고시액을 확인한다. `[모델 지식 — 검증 필요]`

국가법령정보 조회:
```
mcp__korean-law__get_law_text("최저임금법 제6조")
```

**시간급 환산 방법:**

```
시간급 환산액 = 월 통상임금 산입액 ÷ 소정근로시간수 (주휴시간 포함)
```

- 주 40시간 기준: 월 209시간 사용 일반적 `[검토 — 실제 소정근로시간 확인]`
- 주 소정근로시간이 다르면 실제 시간 기준으로 환산

**최저임금 산입 범위 (현행 최저임금법 — 국가법령정보 조회 필수):**
- 매월 1회 이상 정기적으로 지급하는 임금: 원칙 산입
- 상여금: 매월 지급 여부·산입 범위 특칙 확인 `[검토 — 현행 최저임금법 조문 대조 필요]`
- 복리후생비(식대, 교통비 등): 산입 범위 특칙 확인 `[검토]`
- 연장·야간·휴일 가산수당: 산입 제외

`[최저임금법 현행 조문 대조 — 검증 필요]` — 산입 범위는 개정이 있었고 앞으로도 변경될 수 있다.

## §G — 연차유급휴가 발생·사용·수당 체크

국가법령정보 조회:
```
mcp__korean-law__get_law_text("근로기준법 제60조")
mcp__korean-law__get_law_text("근로기준법 제61조")
```

### 연차 발생 기준 (§60) `[모델 지식 — 검증 필요]`

| 근속 기간 | 발생 기준 | 일수 |
|---|---|---|
| 1년 미만 | 개근한 달의 다음 달 1일씩 | 최대 11일 |
| 1년 이상 | 출근율 80% 이상 시 | 15일 |
| 3년 이상 | 2년마다 1일 가산 | 최대 25일 |

- 출근율 산정: 출근 일수 / 소정근로일수 × 100
- 산재·육아휴직·출산휴가 기간은 출근 간주 `[검토 — 사실관계 확인]`

### 연차사용촉진 (§61) — 미사용수당 면제 요건

촉진 절차를 적법하게 이행하면 사용자는 미사용 연차에 대한 금전 보상 의무를 면한다.

**절차 요건 (현행 §61 — 국가법령정보 조회 확인):**
1. **1차 통보:** 연차 소멸 6개월 전, 미사용 일수 서면 통보 + 사용 시기 지정 요청
2. **2차 통보:** 1차 통보 후 사용자가 지정하지 않은 경우, 소멸 2개월 전, 사용자가 시기 지정 통보
3. 통보는 서면 또는 서면에 갈음하는 방식 (이메일·전자문서 가부는 취업규칙·판례 확인)

`[모델 지식 — 검증 필요]` — 촉진 절차 요건은 판례 적용이 엄격하다. 절차 미비 시 면제 불가.

**촉진 절차 체크리스트:**
- [ ] 1차 통보 시점이 소멸 6개월 전인가?
- [ ] 1차 통보에 미사용 일수가 명시되어 있는가?
- [ ] 근로자가 시기를 지정하지 않은 경우 2차 통보를 했는가?
- [ ] 2차 통보 시점이 소멸 2개월 전인가?
- [ ] 통보 방식이 서면 요건을 충족하는가?

### 미사용 연차수당 산정

```
연차수당 = 통상임금 1일치 × 미사용 연차일수
```

- 통상임금 1일치 = 통상임금 시급 × 소정근로시간 (1일)
- 소멸시효: 3년 (§49) `[검토]`

## §H — 임금체불 진정 절차

**적용 국면:** 사용자가 임금·퇴직금을 미지급한 경우.

국가법령정보 조회:
```
mcp__korean-law__get_law_text("근로기준법 제109조")
mcp__korean-law__get_law_text("근로기준법 제49조")
```

### 처벌 조항 (§109) `[모델 지식 — 검증 필요]`

임금체불은 형사처벌 대상: **3년 이하 징역 또는 3천만원 이하 벌금**

### 소멸시효

임금채권: **3년** (§49) — 사유 발생일부터 기산 `[모델 지식 — 검증 필요]`

### 진정 절차

1. **고용노동부 지방관서 진정 접수**
   - 방문 / 우편 / 온라인(고용노동부 민원마당)
   - 근로감독관 조사 배정
   - 사실관계 조사·출석 요구

2. **사용자에 대한 시정 지시**
   - 자진 시정 기간 부여
   - 미이행 시 사법 처리(검찰 송치) 또는 과태료

3. **형사고소** — 진정과 병행 또는 단독 가능
   - 체불임금 명세, 근로계약서, 임금명세서, 출퇴근 기록 등 증거 첨부

4. **소액체당금·체당금 제도**
   - 사업주 파산·폐업 등 경우 근로복지공단에서 체당금 선지급 제도 `[검토 — 요건 확인]`

5. **민사 소송** — 노동청 진정과 별개로 가능

**노동위 경로:** 임금체불 자체는 노동위 관할이 아님. 부당해고·부당노동행위 → `/employment-legal:labor-committee-procedure` 핸드오프.
