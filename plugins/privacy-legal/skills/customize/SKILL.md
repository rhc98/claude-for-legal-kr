# /customize

## 실행 조건

사용자가 `/privacy-legal:customize` 입력. 콜드스타트 인터뷰 전체 재실행 없이, YAML 수동
편집 없이 개인정보 프로파일의 무엇 — 위험 성향·에스컬레이션 담당자·DPA 입장·PIA 섹션·권리행사
타임라인 — 을 변경하고 싶음.

## 할 일

1. **config 읽기.** `~/.claude/plugins/config/claude-for-legal/privacy-legal/CLAUDE.md`
   (와 한 단계 위 `~/.claude/plugins/config/claude-for-legal/company-profile.md`) 읽기.
   플러그인 config 없거나 `[PLACEHOLDER]` 값 남아 있으면:

   > 셋업을 아직 안 하셨습니다. 먼저 `/privacy-legal:cold-start-interview` 실행 —
   > customize는 이미 가진 프로파일을 조정하기 위한 것.

2. **맞춤화 가능 맵 보여주기.** 프로파일에 있는 것을 그룹화, 현재 값의 한 줄 요약과 함께:

   - **회사 / 우리는 누구인가** — 회사명·업종·관할·단계·실무 환경·개인정보처리자/수탁자
     orientation *(모든 12 플러그인 공유 — `company-profile.md` 통해 변경 전파)*
   - **위험 성향** — 보수·중간·적극, 각각의 의미 (수탁자 의무·국외이전·보유 영향)
   - **인력** — CPO·개인정보팀·엔지니어링 연락·외부 변호사·에스컬레이션 체인
   - **위·수탁계약 플레이북** — 수탁자 변경 통지·파기·감사·손해배상·국외이전 입장 (수탁자 측·위탁자 측)
   - **개인정보처리방침 약속** — `/policy-monitor`가 실무 대조 감시하는 약속
   - **PIA 하우스 스타일** — 섹션 순서·위험 점수·이해관계자 framing·의무 평가 트리거
   - **권리행사 절차** — 본인확인·regime별 법정 기한·면제 적용·템플릿 응답 구조
   - **워크플로우** — 인테이크 경로·매터 워크스페이스·policy-monitor sweep cadence
   - **통합** — 문서 저장소·개인정보 도구·Slack·**법망 MCP** 상태·폴백

3. **무엇을 바꿀지 묻기.**

   > 무엇을 조정할까요? 섹션을 고르거나 자기 말로 변경 설명.

4. **변경 수집.** 사용자가 섹션을 고르면 그 섹션의 현재 값을 보여주고 새 값 묻기. 사용자가
   자유 형식으로 설명하면 어느 섹션이 영향인지 식별하고 mapping 확인.

5. **변경 전 검증.** 변경에 법적 인용·임계점·기한이 있으면 법망 대조 검증:
   ```
   mcp__beopmang__law get "[조항]"
   ```
   인용이 안 맞으면 사용자에게 확인 ("당신은 X라 하셨는데, 법망의 현행 텍스트는 Y. 어느
   것으로 갈까요?").

6. **변경 적용.** CLAUDE.md의 해당 섹션을 갱신. 변경 전후 diff 보여주기.

7. **확인 + 검증 로그 기록.**
   ```
   [YYYY-MM-DD] customize - [성명]이(가) [섹션]에서 [구체적 변경]
   ```
   `~/.claude/plugins/config/claude-for-legal/privacy-legal/verification-log.md`에 추가.

## 흔한 맞춤화 패턴

**위험 성향 변경:** "보수에서 중간으로" → 영향 받는 모든 디폴트 다시 보여주고 사용자
확인 받기 (DSAR 본인확인 엄격도, 국외이전 트랙 선호 등).

**에스컬레이션 담당자 변경:** 옛 담당자에서 새 담당자로 — 단순 이름 교체. 매트릭스 전체
보여주기.

**위·수탁계약 입장 단 한 줄 변경:** "재위탁 일괄 사전동의에서 매번 동의로" — 영향: 모든
미래 DPA 검토.

**처리방침 commit 갱신:** "보유 기간 1년에서 2년으로" → policy-monitor 트리거 — 정책
문서 갱신 필요? 사용자에게 묻기.

**법망 MCP 추가·제거:** `.mcp.json` 직접 편집 안내 또는 자동화.
