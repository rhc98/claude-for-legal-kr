# 회사법무 currency watch — 자주 바뀌는 영역

> 회사법무는 **임계점·금액 기준이 매년 갱신**되고 상법·자본시장법 개정이 잦다. 아래 영역에 의존하는
> 분석은 모델 지식에 기대기 전 **법망 또는 1차 소스를 먼저 조회**한다(공유 가드레일 "현행성 트리거").
> 구체 수치는 하드코딩하지 않는다 — 이 파일은 "무엇을 다시 확인해야 하는가"의 목록이지 수치표가 아니다.

## 매년·수시 갱신 (반드시 시점 조회)

| 영역 | 무엇이 바뀌나 | 1차 소스 |
|---|---|---|
| **대규모기업집단 지정** | 공시대상·상호출자제한 기업집단 자산 기준·지정 현황 (매년 5월 전후 공정위 지정) | 공정거래위원회 고시·보도자료 (법망 부분 커버 → `[공정위 자료 — 직접 검증]`) |
| **외부감사 대상** | 자산·매출·부채·종업원 임계점 (외감법 시행령) | 법망 `외부감사법 시행령` |
| **기업결합신고 기준** | 신고대상 회사 자산·매출 규모, 간이/일반 심사 기준 | 법망 `공정거래법 제11조` + 시행령 |
| **상장사 특례 기준** | 사외이사·감사위원회 의무 적용 자산 기준 (상법 시행령) | 법망 `상법 시행령` (§542-8 관련) |
| **소규모회사 기준** | 자본금 10억 기준 등 특례 적용 한계 | 법망 `상법 제363조·제383조·제409조` |
| **공시 서식·기한** | 수시공시·주요사항보고·정기보고 서식·기한 개정 | 한국거래소 공시규정(자율규정 → `[거래소 규정 — 직접 검증]`), 금융위·금감원 |

## 계류·시행 예정 (집행 자세·시행일 확인)

- **상법 개정 동향** — 이사의 충실의무 대상·전자주주총회·집중투표·감사위원 분리선임 등 지배구조 관련
  개정·개정안이 반복 논의된다. 인용 전 시행 여부·시행일 확인. `[모델 지식 — 검증 필요]`
- **자본시장법·공시 개정** — 공정공시·내부자거래·소유상황보고 관련 개정.
- **공정거래법** — 기업결합신고·내부거래·일감몰아주기 규제 개정.

## 데이터 갭 재확인 (법망 미커버 — 사용자 제공 필요)

- 공정위 **기업결합 심사 의결례** → `[공정위 의결 — 직접 검증 필요]`
- 금융위·증선위·금감원 **공시/제재 의결** → `[금융당국 의결 — 직접 검증 필요]`
- 한국거래소 **상장규정·공시규정** → `[거래소 규정 — 직접 검증 필요]`
- 상업등기 **등기선례·등기예규** → `[등기선례 — 검증 필요]`

## 사용 패턴

스킬이 위 영역에 닿는 분석을 할 때:
1. 법망에서 현행 조문·시행령 조회 (임계점·금액·기한)
2. 법망 미커버 영역(공정위 의결·거래소 규정 등)은 사용자에게 자료 요청 + 해당 태그
3. 검증한 항목은 `verification-log.md`에 기록 (다음 사람이 재검증 안 하도록)
