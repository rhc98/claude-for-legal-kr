# ip-legal — 한국 지식재산 플러그인

> [Anthropic claude-for-legal `ip-legal`](https://github.com/anthropics/claude-for-legal)의
> **한국 법체계 포팅**. 국가법령정보 MCP(`https://mcp.gomdori.app/law`)로 특허법·상표법·디자인보호법·저작권법·
> 부정경쟁방지법·발명진흥법 위에서 동작.

상표 클리어런스·자유실시(FTO) 트리아지, 발명 특허성 스크리닝, 침해 트리아지·대응, 경고장·저작권 중단요청
초안, 오픈소스 컴플라이언스, IP 계약조항 검토, 직무발명·영업비밀, 등록·갱신 기한 추적을 다룬다.

**모든 산출물은 변호사·변리사 검토 전제의 초안이다 — 인용·플래그·게이트를 거치되 법적 결론이 아니다.**

---

## 🚧 현재 상태: 포팅 진행 중

상세 진행은 [`skills/ROADMAP.md`](skills/ROADMAP.md) 참조. 상류 11스킬 + KR 신규 2스킬
(`employee-invention`·`trade-secret-protection`) + `ip-renewal-watcher` 에이전트를 Phase A~E로 포팅하고,
**Phase F(E2E 런타임 검증)는 국가법령정보 MCP 복구 후** 진행한다. 국가법령정보 다운 기간에는 모든 조문·판례 인용을
`[검증 필요]`/`[검증 필요-핀포인트]` 태그로 출력한다.

상류는 **미국 IP(USPTO·DMCA·Lanham Act·특허 §101/102/103)**가 전제다. 한국 포팅의 핵심 재맥락화:

- USPTO 특허·상표 검색 → **특허청(KIPO) / KIPRIS** (⚠️ 국가법령정보 미커버 — 사용자 제공)
- 특허요건 §101/102/103 → **특허법 §29**(신규성·진보성)·**§33**
- **DMCA notice & takedown** → **저작권법 §103**(복제·전송 중단요청)·OSP 책임제한 + 정보통신망법
- Cease & desist → **경고장 / 내용증명** (부당경고 역공 위험 인식)
- Lanham Act 혼동 → **상표법 §34·§108** + **부정경쟁방지법 §2** overlay
- Trade secret(DTSA) → **부정경쟁방지법** + KIPI 영업비밀 원본증명 (KR 신규 스킬)
- Employee invention → **발명진흥법 §10·§15 직무발명·정당한 보상** (KR 신규 스킬, 상류 미커버)

## 누구를 위한 것인가

| 역할 | 주 사용 영역 |
|---|---|
| **사내 IP·법무** | 클리어런스·FTO·침해·계약·포트폴리오·직무발명 |
| **특허법인·변리사** | 출원·심판 전략·클리어런스·FTO·포트폴리오 |
| **법무법인 IP** | 침해·라이선스·집행·경고장·영업비밀 |
| **브랜드·콘텐츠 담당** | 상표 클리어런스·저작권 중단요청·OSS |

## 첫 실행

```
/ip-legal:cold-start-interview
```

권리 영역 mix·등록 footprint·집행 입장·에스컬레이션 인터뷰. 결과는
`~/.claude/plugins/config/claude-for-legal-kr/ip-legal/CLAUDE.md`에 저장. 플러그인 업데이트에도 유지.

## 통합

`.mcp.json`에 포함:

- **국가법령정보(korean-law)** — 한국 법령·판례 조회·검증. **1순위.** 없으면 모든 인용이 `[모델 지식 — 검증 필요]`.
- **Slack** — 갱신 기한·침해 대응 알림·에스컬레이션
- **Google Drive** — 경고장 선례·라이선스 계약·보상규정·명세서 로드
- **Box** — IP 계약·라이선스·실사 문서 관리

⚠️ **특허청 KIPRIS는 MCP가 없다** — 상표·특허·디자인 검색·등록원부는 사용자가 검색결과를 직접 제공한다.

## 국가법령정보 데이터 갭 (ip 특유)

국가법령정보가 커버하지 않는 IP 자료는 사용자 제공 + 태그로 처리한다([`../../docs/DATA_GAPS.md`](../../docs/DATA_GAPS.md)):

- **특허청 KIPRIS**(상표·특허·디자인 등록/출원 원부) → `[KIPRIS — 직접 검증 필요]`
- **특허심판원(IPTAB) 심결** → `[심판원 심결 — 직접 검증 필요]`
- **특허청 수수료·존속기간·갱신기한 고시** → `[특허청 고시 — 직접 검증 필요]`
- **한국특허정보원(KIPI) 영업비밀 원본증명** → `[KIPI 원본증명 — 직접 검증 필요]`
- **한국저작권위원회 등록·조정례 / 문체부 OSP 고시** → `[저작권위 등록 — 직접 검증 필요]`
- ✓ 잘 커버: 특허법·실용신안법·상표법·디자인보호법·저작권법·부정경쟁방지법·발명진흥법 + 특허법원·대법원 판례
