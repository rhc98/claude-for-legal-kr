# 법망 미커버 데이터 — 알려진 갭

> 법망(Beopmang)은 법령·판례·헌재결정 등 1차 법원(法源)은 광범위하게 커버하지만, 한국
> 법무 실무에 중요한 일부 행정 결정례·외국법은 포함하지 않는다. 이 문서는 그 갭을
> 명시하고, 각 갭을 어떻게 처리할지 안내.

## 갭 카테고리

### 1. 행정 결정례

**개인정보보호위원회 (PIPC) 결정례**
- 법망 미커버
- 사이트: `https://www.pipc.go.kr` → 결정사항·자료실
- 형식: PDF 결정문, 사이트 검색 인터페이스
- 빈도: PIPA 시행조치·과징금 결정이 핵심 — 실무 영향 큼
- **임시 처리:** 사용자가 결정문 paste·URL 첨부. 모든 인용에 `[PIPC 결정 — 직접 검증 필요]` 태그.

**개인정보 분쟁조정위원회 결정례**
- 법망 미커버
- KISA 운영, 별도 사이트
- 형식: 결정 요약 + 일부 결정문 공개
- 빈도: 정보주체 권리행사 분쟁 핵심
- **임시 처리:** 사용자가 결정 요약 paste·링크.

**공정거래위원회 의결례**
- 법망 미커버 (의결례는 분리)
- 사이트: `https://www.ftc.go.kr` → 의결·심결
- 형식: 사이트 검색, 의결서 PDF
- 빈도: 표시광고법·전자상거래법·하도급법 분야에 개인정보 issue 포함
- **임시 처리:** commercial-legal·corporate-legal 플러그인에서 사용자 paste.

**방송통신위원회 의결례**
- 법망 미커버
- 사이트: `https://www.kcc.go.kr` → 의결사항
- 빈도: 정보통신망법 위반 시정조치
- **임시 처리:** 사용자 paste·링크.

**금융감독원·금융위 결정례**
- 법망 부분 커버 (행정규칙 일부)
- 사이트: 각 위원회 사이트
- 빈도: 금융권 개인정보·신용정보 처리에 핵심
- **임시 처리:** corporate-legal·privacy-legal 플러그인에서 분야별.

**노동위원회 판정례**
- 법망 미커버
- 중앙·지방 노동위, 사건검색 시스템
- 사이트: `https://www.nlrc.go.kr`
- 빈도: 부당해고 구제 신청·차별 시정 신청 핵심
- **임시 처리:** employment-legal 플러그인.

**특허심판원(IPTAB) 심결**
- 법망 미커버 (법원 판례는 커버, 심판원 심결은 분리)
- 사이트: 특허심판원 → 심판정보, KIPRIS 심판 검색
- 형식: 심결문 PDF, 사건검색
- 빈도: 무효심판·권리범위확인심판·거절결정불복심판 — ip-legal 침해·무효 분석 핵심
- **임시 처리:** ip-legal 플러그인에서 사용자 paste, `[심판원 심결 — 직접 검증 필요]` 태그.

### 1-A. 지식재산 등록·출원 데이터 (특허청 KIPRIS — ip-legal 핵심 갭)

법망은 **법령·판례**를 커버하지만, **권리의 등록·출원 현황 자체**(상표·특허·디자인 등록원부, 출원
경과정보, 선행기술·선등록 검색)는 특허청 KIPRIS의 영역이며 **MCP 커넥터가 없다**. ip-legal의
클리어런스·FTO·포트폴리오·갱신이 이 데이터에 의존한다.

**특허청 KIPRIS (상표·특허·디자인 등록/출원 원부·검색)**
- 법망 미커버, 전용 MCP 없음
- 사이트: `https://www.kipris.or.kr` (검색), `https://www.patent.go.kr`(특허로 — 수수료·기한·연차료)
- 형식: 웹 검색 인터페이스, 등록원부, 공보
- 빈도: 클리어런스(선등록 상표)·FTO(차단특허)·포트폴리오(등록현황)·갱신 — ip-legal 상시
- **임시 처리:** 사용자가 KIPRIS 검색결과·등록원부를 캡처·paste. 모든 항목 `[KIPRIS — 직접 검증 필요]` 태그.
  존재하지 않는 선등록·선행기술을 절대 지어내지 않는다(침묵 보완 금지).

**특허청 수수료·존속기간·갱신기한 고시**
- 법망 부분/미커버 (특허청 고시·요율표)
- 사이트: 특허로(patent.go.kr) 수수료 안내
- 빈도: 연차등록료·상표 갱신·디자인 존속기간 산정 — portfolio·ip-renewal-watcher 핵심
- **임시 처리:** `[특허청 고시 — 직접 검증 필요]` 태그.

**한국특허정보원(KIPI) 영업비밀 원본증명**
- 법망 미커버
- 운영: 한국특허정보원 영업비밀보호센터
- 빈도: trade-secret-protection 스킬 — 원본증명 등록 여부·절차
- **임시 처리:** `[KIPI 원본증명 — 직접 검증 필요]` 태그.

**한국저작권위원회 등록·분쟁조정례 / 문체부 OSP 고시·서식**
- 법망 미커버
- 사이트: 한국저작권위원회, 문화체육관광부
- 빈도: takedown(저작권법 §103 중단요청 서식)·저작권 등록 — ip-legal
- **임시 처리:** `[저작권위 등록 — 직접 검증 필요]` 태그.

### 2. 법망 부분 커버 영역

**행정규칙 (고시·훈령·예규)**
- 법망 부분 커버 — 약 24,000건 (전체의 일부)
- 새 고시는 동기화 지연 가능 (매주 토요일)
- 1차: 법제처 행정규칙정보시스템

**자치법규 (지자체 조례·규칙)**
- 법망 커버 — 약 159,000건
- 단, 일부 조례 본문 일부만 색인 가능성
- 1차: 자치법규정보시스템 (`https://www.elis.go.kr`)

### 3. 외국법

**EU법 (GDPR·EU AI Act 등)**

- **임시 처리:** 외국법 변호사 자문 또는 웹 검색 + `[외국법 — 외국 변호사 검증 필요]` 태그

**미국법 (CCPA·HIPAA·연방법 등)**
- 법망 미커버

**기타 외국법**
- 일본 APPI, 중국 PIPL, 싱가포르 PDPA 등 — 법망 미커버
- 1차: 각국 정부 사이트, 또는 외국 법무 데이터베이스 (Westlaw·LexisNexis 등)

## 갭 처리 표준 패턴

스킬에서 법망 미커버 자료를 인용해야 할 때:

```markdown
1. 법망 검색 시도
2. 빈 결과 또는 부적합 결과 시 사용자에게 알림:
   "이 [PIPC 결정 / 공정위 의결 / 노동위 판정]은 법망에 색인되지 않습니다.
   결정문 URL 또는 PDF를 paste해주시면 인용에 반영하겠습니다."
3. 사용자 제공 받으면 인용에 `[사용자 제공 — 직접 검증 필요]` 또는
   해당 출처별 태그 (`[PIPC 결정 — 직접 검증 필요]` 등) 부착
4. 출력에 해당 데이터 갭 명시
```

## 향후 데이터 확장 옵션

### 옵션 A: 별도 데이터 인덱스 구축

지원 가능한 contributor가 행정 결정례 데이터 인덱스를 별도로 구축:

- PIPC·공정위·노동위 등 결정문 RSS·웹 스크래핑 (해당 사이트 약관 준수 필요)
- Embedding 기반 검색 인덱스 자체 구축
- 별도 MCP 서버로 노출
- 본 플러그인의 `.mcp.json`에 추가

이 옵션은 상당한 엔지니어링 + 법적 검토 (스크래핑 약관) 필요.

### 옵션 B: 법망과 협업

법망 측에 행정 결정례 색인 추가 요청. 법망이 1차 데이터 확장 방향성 있으면 가능성.

### 옵션 C: 수동 큐레이션

자주 인용되는 핵심 행정 결정례를 본 레포의 `references/` 디렉토리에 수동 큐레이션. 한계는
유지 노력 + 결정례가 매주 늘어남.

## 사용자 행동 안내

이 데이터 갭으로 인해:

1. **PIPC 결정·공정위 의결을 인용하는 산출물은 사용자 직접 검증 필수**
2. **외국 정보주체 대상 처리는 외국법 변호사 자문 권장**
3. **수동 paste한 자료는 단일 정보주체용** — 다른 매터에 재이용 시 재확인
